# 第五章 回测结果解读

> 回测是量化交易的实验室。但实验室里的成功，不等于战场上的胜利。
> 本章将帮你读懂回测报告中的每一个数字，识别那些让你自欺欺人的陷阱。

## 5.1 各指标含义与优秀标准

回测报告里塞满了各种指标。下面这张表是你需要关注的**核心指标清单**，以及它们的"优秀"分数线。

### 5.1.1 核心指标速查表

| 指标 | 英文名 | 计算方式 | 优秀标准 | 可接受线 | 危险线 | 一句话解释 |
|------|--------|----------|----------|----------|--------|------------|
| **年化收益率** | Annualized Return | `(终值/初值)^(252/天数) - 1` | > 20% | > 10% | < 0% | 每年平均赚多少 |
| **夏普比率** | Sharpe Ratio | `(年化收益 - 无风险利率) / 年化波动率` | **> 1.0** | > 0.5 | < 0 | 每单位风险换多少超额收益 |
| **最大回撤** | Max Drawdown | `max((峰值 - 谷值) / 峰值)` | **< 15%** | < 20% | > 30% | 最惨的时候亏了多少 |
| **Calmar比率** | Calmar Ratio | `年化收益率 / 最大回撤` | **> 1.0** | > 0.5 | < 0.3 | 收益与最大回撤的性价比 |
| **胜率** | Win Rate | `盈利交易次数 / 总交易次数` | > 55% | > 50% | < 45% | 交易赚钱的概率 |
| **盈亏比** | Profit/Loss Ratio | `平均单笔盈利 / 平均单笔亏损` | > 2.0 | > 1.5 | < 1.0 | 赚一次够亏几次 |
| **Alpha** | Alpha | `实际收益 - (Beta × 市场收益)` | > 10% | > 0% | < 0% | 你的策略的真本事 |
| **Beta** | Beta | `Cov(策略, 市场) / Var(市场)` | 0.5~1.0 | < 1.5 | > 2.0 | 跟市场的亲密度 |
| **索提诺比率** | Sortino Ratio | `(年化收益 - 无风险利率) / 下行波动率` | **> 1.5** | > 1.0 | < 0 | 只惩罚下跌的夏普 |
| **信息比率** | Information Ratio | `超额收益 / 跟踪误差` | > 0.5 | > 0 | < 0 | 跑赢基准的稳定性 |
| **换手率(年化)** | Annual Turnover | `年交易额 / 平均持仓市值` | < 10x | < 20x | > 50x | 越高手续费越疼 |

### 5.1.2 各指标深度解读

#### 年化收益率 (Annualized Return)

这是你最直观的回报数字。但要记住：**高收益必然伴随高风险**。如果你看到一个年化 50% 的策略，请立刻去检查它的最大回撤——大概率超过 40%。

```
年化收益率 = (期末净值 / 期初净值) ^ (252 / 回测天数) - 1
```

其中 252 是 A 股每年大约的交易日数。

#### 夏普比率 (Sharpe Ratio)——"性价比之王"

夏普比率是量化领域最重要的单一指标。它回答了核心问题：**你每承担一单位波动（风险），换来了多少超额收益？**

| 夏普范围 | 评价 | 现实含义 |
|----------|------|----------|
| 0 ~ 0.5 | 一般 | 比存银行强点，但不多 |
| 0.5 ~ 1.0 | 良好 | 有超额收益，但波动较大 |
| **1.0 ~ 2.0** | **优秀** | **机构级表现，值得真金白银测试** |
| 2.0 ~ 3.0 | 卓越 | 顶级对冲基金水平 |
| > 3.0 | 可疑 | 建议先检查是否有未来信息泄露 |

> **经验之谈**：如果某策略的样本内夏普 > 3.0，实盘夏普通常会在 0.5~1.0 之间。打三折是一个现实的预期。

#### 最大回撤 (Max Drawdown)——"你的心理承受测试"

最大回撤告诉你：如果运气最差，在最高点买入、最低点卖出，你会亏百分之多少。

| 最大回撤 | 普通人心理反应 |
|----------|----------------|
| < 10% | "还行，波动正常" |
| 10% ~ 20% | "有点慌，但还能忍" |
| 20% ~ 30% | "我要不要止损？" |
| 30% ~ 50% | "量化果然是骗人的" |
| > 50% | "已删软件，再见股市" |

**关键事实**：大多数人高估了自己的风险承受能力。回测时觉得 30% 回撤无所谓，实盘中 15% 就开始失眠。**建议把你能接受的回撤数字除以 2**，按那个标准来评估策略。

#### Calmar比率——"收益回撤比"

```
Calmar = 年化收益率 / 最大回撤
```

举例：年化 20%，最大回撤 15%，Calmar = 1.33。这意味着你每承受 1% 的回撤痛苦，换回 1.33% 的年化收益。

Calmar > 1.0 是及格线。低于这个数，说明你承受的痛苦和获得的回报不成比例。

#### 胜率 (Win Rate) 与盈亏比

这两个指标必须放在一起看。两者存在经典的"跷跷板"关系：

| 策略类型 | 胜率 | 盈亏比 | 典型例子 |
|----------|------|--------|----------|
| 趋势跟踪 | 35%~45% | 2.5~5.0 | 海龟交易法 |
| 均值回归 | 55%~65% | 1.2~1.8 | 布林带策略 |
| 高频做市 | 50%~55% | 1.01~1.05 | 依靠大数定律 |

**数学真相**：期望收益 = 胜率 × 平均盈利 - (1-胜率) × 平均亏损。只要期望值为正，胜率低也能赚钱。趋势跟踪策略胜率往往不到 40%，但盈亏比高达 3:1，最终仍然盈利。

#### Alpha 与 Beta——"天赋 vs 风口"

```
策略收益 = Alpha + Beta × 市场收益 + 残差
```

- **Beta** = 你在"吃市场红利"的程度。Beta = 1 意味着市场涨 10%，你也涨 10%。Beta = 0 意味着你完全不依赖大盘。
- **Alpha** = 剔除市场波动后，你的策略**自身**创造的价值。这才是你真正的能力。

**残酷真相**：很多策略看起来收益高，拆开一看 Beta = 1.2，Alpha 接近 0。这意味着你只是在牛市里加杠杆，熊市来了会加倍亏回去。

#### 索提诺比率 (Sortino Ratio)

索提诺是夏普的"变体"。夏普把所有波动都视为风险，但索提诺认为：**向上波动不是风险，向下波动才是**。

```
索提诺 = (年化收益 - 无风险利率) / 下行标准差
```

索提诺比夏普更适合评估你真正关心的"亏钱风险"。一般来说，索提诺 > 1.5 才算好策略。

## 5.2 回测常见陷阱——你最容易骗自己的 5 种方式

### 陷阱一：幸存者偏差 (Survivorship Bias)

**症状**：你用 2024 年的沪深 300 成分股，回测 2019~2023 年的策略。表现极好。

**问题**：2019 年的沪深 300 成分股和 2024 年的不完全一样。那些被剔除的股票——通常是因为跌得太惨——根本没有出现在你的回测里。你只用"活下来的"股票做了测试。

**后果**：回测收益被**系统性高估**了，因为你排除了表现最差的股票。

**解决方案**：
- 每个时间点只使用**当时**的成分股
- 使用包含退市股票的完整数据库
- 如果只有当前成分股数据，对回测结果自行打 7~8 折

### 陷阱二：未来信息泄露 (Future Information Leak / Look-Ahead Bias)

这是**最常见、最隐蔽、最致命**的错误。

**症状 A**：用当天的收盘价来决定当天的交易。比如"如果今日收盘价 > 5 日均线，则今日买入"。但在现实中，当你看到收盘价时，交易已经结束了。

**症状 B**：用财报公布后的数据来训练模型，却把模型应用在财报公布之前的时间点。

**症状 C**：在计算 20 日均线时，把"今天"也算进去了，然后根据这个均线在"今天"交易。

**后果**：回测结果完全不可靠，实盘必然亏损。

**解决方案**：
- 所有信号计算**严格使用 T-1 及之前的数据**
- 交易执行在 T 日开盘价（或 VWAP）
- 建立数据时间戳检查机制，每笔交易确认"决策时能获取这些数据吗？"

```python
# 错误示范：用今日收盘价做今日决策
signal = close_price.pct_change()  # 包含了当日信息
position = signal.apply(lambda x: 1 if x > 0 else 0)  # 不可能做到

# 正确做法：用昨日信号决定今日交易
signal = close_price.pct_change().shift(1)  # 滞后一天
position = signal.apply(lambda x: 1 if x > 0 else 0)  # 可行
```

### 陷阱三：过拟合 (Overfitting)——"量化师的毒药"

**症状**：你的策略在训练集上夏普 3.5，测试集上夏普 0.2。

**问题**：你使用了 15 个参数（均线参数、止损比例、仓位大小、选股条件……），通过反复调整，找到了**恰好**让历史数据看起来最好的一组。但这组参数捕捉的是历史的噪音，而非真正的规律。

**判断自己是否过拟合的 5 个问题**：
1. 你的参数数量是否超过 5 个？
2. 你是否针对回测结果"微调"过参数 10 次以上？
3. 策略逻辑是否复杂到你无法用一句话向朋友解释？
4. 去掉最好的 3 笔交易后，策略是否还盈利？
5. 略微改变参数（如均线从 20 变成 21），收益是否大幅变化？

如果前三个问题你都回答"是"，你很可能在过拟合。

**解决方案**：
- 减少参数数量（少于 5 个是理想状态）
- 使用样本外测试（见 5.3 节）
- 参数敏感性分析：参数 ±20%，收益变化不应该超过 30%
- **奥卡姆剃刀原则**：简单的策略比复杂的策略更可能在实盘中存活

### 陷阱四：手续费与滑点忽略

**症状**：回测年化 30%，扣除万三手续费 + 千一滑点后，只剩 8%。

不同频率的策略对手续费的敏感度完全不同：

| 策略类型 | 年换手率 | 手续费影响 | 是否致命 |
|----------|----------|------------|----------|
| 低频选股（月换仓） | 12x | 约 0.3%~0.5% | 一般 |
| 中频轮动（周换仓） | 52x | 约 1.5%~3% | 需关注 |
| 高频日内 | 200x+ | 约 10%~30% | **致命** |
| 做市策略 | 1000x+ | > 50% | **绝对致命** |

**必算公式**：

```
实际净收益 ≈ 回测毛收益 - (年换手率 × 单边费率 × 2) - 滑点估算
```

其中 A 股单边费率约 0.025%（佣金）+ 0.05%（印花税，仅卖出）= 买入 0.025%，卖出 0.075%。再加上约 0.1% 的滑点，每完成一次买卖的实际成本约为 **0.2%~0.3%**。

### 陷阱五：前视偏差在因子计算中的变种

除了"用今天数据决定今天交易"，还有一些更隐蔽的前视偏差：

- **财务数据发布延迟**：Q1 财报最早 4 月底才公布，但你用的数据库里 Q1 数据标签是 3 月 31 日
- **股票更名/合并**：某股票在 2023 年发生合并，回测时 2022 年用的就是合并后的数据
- **指数成分调整提前知晓**：6 月宣布的调整，12 月才生效，但 6 月的价格已经反映了

## 5.3 如何判断回测可信——4 项"压力测试"

### 测试一：参数稳定性检验

**核心思想**：一个好策略不应该只在一个特定参数值下有效。

**操作方法**：
1. 对每个参数，在 ±50% 范围内取 10 个等距值
2. 分别回测，记录夏普比率
3. 绘制"参数-夏普"曲线

**判断标准**：
- 曲线平滑（没有尖锐的峰值）：可信
- 曲线有一个突兀的尖峰，其他地方都是负数：**过拟合**

```
夏普
3.0 |          *
2.0 |        *   *
1.0 |  *   *       *   *
0.0 |*                   *
    +---+---+---+---+---+--- 参数值
    10  14  18  22  26  30

OK：峰值不突兀，周围区域也不错

夏普
3.0 |          *
2.0 |
1.0 |
0.0 |* * * *       * * * *
    +---+---+---+---+---+--- 参数值
    10  14  18  22  26  30

危险！只在 20 这一档有效，其他都烂
```

### 测试二：不同时间段检验

将回测数据分为几个子区间，单独计算每个区间的夏普：

| 时间段 | 市场环境 | 策略夏普 | 结论 |
|--------|----------|----------|------|
| 2019.01 - 2019.12 | 牛市 | 2.1 | OK |
| 2020.01 - 2020.12 | 震荡 | 1.3 | OK |
| 2021.01 - 2021.12 | 结构市 | 0.8 | 尚可 |
| 2022.01 - 2022.12 | 熊市 | 0.2 | 勉强 |
| 2023.01 - 2023.12 | 震荡下跌 | 0.9 | OK |

**判断标准**：所有子区间的夏普都 > 0，说明策略有跨市场环境的适应能力。如果在某个子区间大幅为负，说明策略只在特定行情下有效。

### 测试三：随机打乱股票检验 (Permutation Test)

**核心思想**：如果你的策略真的捕捉到了规律，那么把股票标签随机打乱后，策略应该失效。

**操作方法**：
1. 将股票收益率的时间序列随机打乱（保持每只股票的时间序列结构，但打乱不同股票之间的对应关系）
2. 在打乱的数据上运行你的策略
3. 重复 100 次
4. 看真实回测的夏普在 100 次随机回测夏普分布中的位置

**判断标准**：如果你的真实夏普比 95% 的随机结果都好（即 p-value < 0.05），说明策略**可能**有真实信号。如果真实夏普落在随机分布中间，说明你的策略和扔硬币没有区别。

### 测试四：样本外测试 (Out-of-Sample Test)

这是**最硬核**的检验，也是在实盘之前必须通过的。

**方法 A：时间分割**
- 训练集：2019.01 - 2022.12（前 70%）
- 验证集：2023.01 - 2023.12（后 15%）——用于调参
- 测试集：2024.01 - 2024.12（最后 15%）——**放手不动，只看结果**

**方法 B：滚动窗口 (Walk-Forward)**
- 用 3 年数据训练 → 预测下 1 个月 → 窗口向前滚动 → 重复
- 这样每个月的预测都是真正的"未来"，从未被模型见过

```python
# 滚动窗口测试伪代码
for test_start in range(first_test_date, last_test_date, step='1M'):
    train_data = data[test_start - 3 years : test_start]
    test_data  = data[test_start : test_start + 1 month]

    model.fit(train_data)
    predictions = model.predict(test_data)
    # 累计记录预测结果

# 最终的权益曲线完全来自样本外预测
```

**判断标准**：样本外夏普 > 样本内夏普 × 0.5 即可接受。样本外夏普 > 1.0 则是优秀的策略。

## 5.4 双均线策略——实战可行性坦诚评估

你的项目中使用双均线策略（Golden Cross / Death Cross）作为入门示例。这是量化世界里被研究最充分的策略之一。以下是坦诚的评估。

### 策略原理

```
短期均线（如 5 日）上穿长期均线（如 20 日）→ 买入（金叉）
短期均线下穿长期均线                → 卖出（死叉）
```

### 优点

| 优点 | 说明 |
|------|------|
| **极其简单** | 只有 2 个参数，几乎不会过拟合 |
| **逻辑透明** | 连不懂量化的人都能理解你在做什么 |
| **趋势捕捉** | 在大趋势行情中表现不错 |
| **低门槛** | 任何平台都能实现，是学习量化最好的起点 |
| **可解释性强** | 每笔交易都有清晰的原因 |

### 缺点（诚实版）

| 缺点 | 严重程度 | 说明 |
|------|----------|------|
| **震荡市亏死** | 高 | 价格反复穿越均线，频繁产生虚假信号 |
| **滞后性** | 高 | 均线是滞后指标，趋势反转后还要几天才确认 |
| **横盘无效** | 高 | 没有趋势 = 没有利润 |
| **不区分行情** | 中 | 牛市和熊市用同一套参数 |
| **被机构碾压** | 高 | 这策略 50 年前就有了，全市场都知道 |
| **散户误用** | 中 | 很多人只看日线，忽略了更大级别的趋势 |

### 实测表现预估（A 股市场）

基于沪深 300 指数 2014~2024 年的历史数据，5/20 双均线策略的**大致**表现：

| 指标 | 大约数值 | 评价 |
|------|----------|------|
| 年化收益 | 6% ~ 12% | 略高于持有不动 |
| 夏普比率 | 0.3 ~ 0.6 | 一般 |
| 最大回撤 | 25% ~ 35% | 较大 |
| 胜率 | 35% ~ 42% | 偏低 |
| Calmar | 0.2 ~ 0.4 | 一般 |
| 跑赢买入持有？ | 有时 | 主要在熊市减仓起作用 |

### 为什么双均线在实盘中比回测更差？

1. **滑点问题**：金叉死叉往往出现在价格快速变动的时候，此时买卖价差扩大
2. **假突破频繁**：A 股日内波动大，可能盘中触发信号又收回来
3. **手续费累积**：震荡市中可能一个月进出好几次，每次 0.25%，一年就是 3%~6% 的成本
4. **心态问题**：连续 5 次假信号后，第 6 次你还敢跟吗？

### 坦白结论

> **双均线策略不适合直接用于实盘交易。但它是学习量化交易最好的"第一行代码"。**
>
> 通过这个策略，你可以学会：如何计算指标、如何生成信号、如何回测、如何评估。这些技能才是真正有价值的东西，而不是双均线本身。
>
> 如果你真的想在实盘中使用均线，请至少做以下改进：
> - 加入成交量确认
> - 加入趋势过滤器（ADX > 25 才允许交易）
> - 加入波动率过滤器（ATR 判断是震荡还是趋势）
> - 使用自适应均线而非固定参数
> - 多时间框架确认（周线定方向，日线找时机）

## 5.5 回测报告自我审查清单

在把一个策略投入实盘之前，请逐条核验：

```
□ 所有信号都只使用 T-1 及之前的数据
□ 交易成本已计算（佣金 + 印花税 + 滑点）
□ 使用了完整的成分股历史（非当前成分股）
□ 考虑了跌停无法卖出、涨停无法买入的情况
□ 策略参数少于 5 个
□ 不同时间段的表现都为正
□ 参数敏感性测试通过（参数 ±20%，表现变化 < 30%）
□ 样本外测试通过（夏普 > 样本内 × 50%）
□ 最大回撤在我的心理承受范围之内（除以 2 之后！）
□ 我知道这个策略在什么市场环境下会失效
```

如果你能诚实地说上面 10 条全部满足，那么这个策略**值得用小资金测试**。如果超过 3 条不能满足，请回到画板继续改进。

---

> **回测能告诉你过去发生了什么，但不能保证未来会重复。**
> 量化交易的核心不是找到"最好的回测曲线"，而是找到"最可能在实盘中存活"的策略。
> 永远用怀疑的眼光看待你的回测结果——这是对自己资金负责的态度。
