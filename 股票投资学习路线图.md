## 🚀 快速开始

### 第一步：投喂第一个提示词

打开 Claude Code，复制粘贴：
```
prompts/part1-基础/M1-股票零基础入门手册.md
```

Claude Code 会自动生成 `docs/manual/01-股票市场零基础入门.md`

### 第二步：依次执行

按 M1→M2→M3→Q1→Q2→Q3→Q4→A1→A2→A3→A4 顺序执行。

## 💡 使用技巧

| 问题 | 解决方法 |
|------|---------|
| Claude Code 生成中断 | 新开对话，把提示词后半部分重新喂入 |
| 代码有 bug | 把报错信息发给 Claude Code，让它修 |
| akshare 安装失败 | Claude Code 会给你替代方案 |
| 没有 GPU 跑 LLM | 用 Ollama + CPU（慢但能用）或免费 API |
| 不理解某个概念 | 单独开对话问 Claude Code 解释 |

## 🎓 进阶路线（学完本课程后）

1. **因子挖掘**：尝试自己发现新因子（用遗传规划算法）
2. **高频数据**：引入分钟级数据做日内策略
3. **强化学习**：用 RL 训练交易智能体
4. **实盘对接**：接入券商 API（如 QMT/Ptrade）
5. **另类数据**：卫星图像、社交媒体情绪、供应链数据
