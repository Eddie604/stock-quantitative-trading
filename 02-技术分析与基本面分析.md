### 第一章 K线图深入解读

#### 1.1 K线的基本构成：实体与影线

一根K线记录了某一时间周期内的四个关键价格：**开盘价（Open）**、**最高价（High）**、**最低价（Low）**、**收盘价（Close）**，合称OHLC。

```
        最高价                          最高价
         │                               │
         │     上影线                     │     上影线
    ┌────┴────┐                     ┌────┴────┐
    │         │                     │         │
    │  实体   │ ← 开盘/收盘         │  实体   │ ← 开盘/收盘
    │         │                     │         │
    └────┬────┘                     └────┬────┘
         │     下影线                     │     下影线
         │                               │
        最低价                          最低价

      阳线（收盘 > 开盘）              阴线（收盘 < 开盘）
      实体通常为红色或空心            实体通常为绿色或实心
```

- **实体**：开盘价与收盘价之间的距离。实体越长，表明多空一方的主导力量越强。
- **上影线**：最高价与实体上沿之间的距离。长上影线表示价格曾冲高但遭遇抛压回落。
- **下影线**：实体下沿与最低价之间的距离。长下影线表示价格曾下探但被买盘托起。

> **一句话总结：实体看力度，影线看分歧。实体长说明方向明确，影线长说明多空博弈激烈。**

#### 1.2 单根K线的经典信号

以下是七种最常见的单根K线形态及其市场含义：

**（1）大阳线**

```
         │
    ┌────┴────┐
    │         │    实体长度 >> 影线长度
    │  大阳   │    收盘远高于开盘
    │         │
    └────┬────┘
         │
```
- **含义**：买方压倒性优势，出现在低位可能开启上涨趋势，出现在高位需警惕加速赶顶。
- **A股实例**：2024年9月24日，受政策组合拳刺激，上证指数单日涨4.15%，多只权重股收出大阳线。

**（2）大阴线**

```
    ┌────┴────┐
    │         │    实体长度 >> 影线长度
    │  大阴   │    收盘远低于开盘
    │         │
    └────┬────┘
         │
```
- **含义**：卖方压倒性优势，高位大阴线是典型反转信号。

**（3）十字星**

```
         │
         │
    ─────┼─────   开盘 ≈ 收盘，实体极小
         │        上下影线长度相当
         │
```
- **含义**：多空力量暂时均衡，是趋势可能反转的预警信号。十字星出现在连续上涨或下跌之后意义最大。

**（4）锤子线**

```
         │
    ┌────┴────┐
    │  小实体  │
    └────┬────┘
         │
         │        下影线 ≥ 实体 × 2
         │        上影线极短或没有
         │
```
- **含义**：出现在下跌趋势末端，价格曾大幅下探但被强势买回，表明下方支撑强劲，是潜在底部反转信号。

**（5）倒锤子**

```
         │
         │        上影线 ≥ 实体 × 2
         │        下影线极短或没有
    ┌────┴────┐
    │  小实体  │
    └────┬────┘
         │
```
- **含义**：出现在下跌趋势末端，价格冲高回落但收盘仍高于开盘，暗示买盘开始试探性入场。

**（6）吊颈线**

```
         │
    ┌────┴────┐
    │  小实体  │    形态与锤子线相同
    └────┬────┘    但出现在上涨趋势末端
         │
         │        下影线 ≥ 实体 × 2
         │
```
- **含义**：形态完全同锤子线，但位于上涨趋势顶部，是警示信号。次日的确认阴线至关重要。

**（7）射击之星**

```
         │
         │        形态与倒锤子相同
         │        但出现在上涨趋势末端
    ┌────┴────┐
    │  小实体  │
    └────┬────┘
         │
```
- **含义**：价格冲高后大幅回落，收盘接近开盘价，表明上方抛压沉重。

> **一句话总结：同一形态在不同位置含义截然不同——锤子线在底部是希望，在顶部是陷阱。**

#### 1.3 K线组合形态

**（1）吞没形态**

看涨吞没：

```
    前一天      后一天
      │           │
  ┌───┴───┐  ┌───┴───┐
  │ 小阴  │  │       │  后一天阳线实体
  └───┬───┘  │ 大阳  │  完全覆盖前一天
      │      │       │  阴线实体
             └───┬───┘
                 │
```

看跌吞没：

```
    前一天      后一天
      │           │
  ┌───┴───┐  ┌───┴───┐
  │ 小阳  │  │       │  后一天阴线实体
  └───┬───┘  │ 大阴  │  完全覆盖前一天
      │      │       │  阳线实体
             └───┬───┘
                 │
```
- **确认条件**：第二根K线实体必须完全包裹第一根实体（影线可忽略），且成交量放大增强信号可靠性。

**（2）早晨之星与黄昏之星**

早晨之星（底部反转）：

```
    Day1      Day2      Day3
     │                    │
 ┌───┴───┐   十字星   ┌───┴───┐
 │ 大阴  │   ───┼───  │ 大阳  │  阴→星→阳，第三根
 └───┬───┘    (跳空)  └───┬───┘  至少收复第一根50%
     │                    │
```

黄昏之星（顶部反转）：

```
    Day1      Day2      Day3
     │                    │
 ┌───┴───┐   十字星   ┌───┴───┐
 │ 大阳  │   ───┼───  │ 大阴  │  阳→星→阴，第三根
 └───┬───┘    (跳空)  └───┬───┘  至少吞没第一根50%
     │                    │
```

**（3）红三兵与三只乌鸦**

红三兵（连续三根稳步上涨的阳线）：

```
  Day1    Day2    Day3
   │       │       │
┌──┴──┐ ┌──┴──┐ ┌──┴──┐
│     │ │     │ │     │  每根收盘均高于前一根
│ 阳  │ │ 阳  │ │ 阳  │  每根开盘均在前一根实体之内
└──┬──┘ └──┬──┘ └──┬──┘
   │       │       │
```

三只乌鸦（连续三根下跌阴线）：

```
  Day1    Day2    Day3
┌──┴──┐ ┌──┴──┐ ┌──┴──┐
│     │ │     │ │     │  每根收盘均低于前一根
│ 阴  │ │ 阴  │ │ 阴  │  每根开盘均在前一根实体之内
└──┬──┘ └──┬──┘ └──┬──┘
   │       │       │
```

> **一句话总结：K线组合的本质是多空力量在时间序列上的渐变——单根K线是快照，组合才是故事。**

### 第二章 均线系统

#### 2.1 移动平均线的数学原理

移动平均线（Moving Average, MA）是过去N个周期收盘价的算术平均，其核心作用是平滑价格波动，揭示趋势方向。

- **SMA（简单移动平均）**：`SMA = (P₁ + P₂ + ... + Pₙ) / N`，每根K线权重相等。
- **EMA（指数移动平均）**：`EMAₜ = α × Pₜ + (1 − α) × EMAₜ₋₁`，其中 `α = 2 / (N + 1)`。EMA赋予近期价格更高权重，对价格变化的反应比SMA更灵敏。

**SMA与EMA的对比：**

| 特性 | SMA | EMA |
|------|-----|-----|
| 灵敏度 | 较迟钝 | 较灵敏 |
| 假信号 | 较少 | 较多 |
| 适用场景 | 中长期趋势判断 | 短期交易信号 |
| 滞后性 | 明显 | 相对较小 |

> **一句话总结：SMA适合看清大方向，EMA适合捕捉入场时机——长线看SMA，短线用EMA。**

#### 2.2 A股常用均线周期

A股市场因交易制度和投资者结构，形成了约定俗成的均线周期体系：

| 周期 | 名称 | 市场含义 |
|------|------|----------|
| MA5 | 周线 | 超短线情绪，股价在MA5上方运行视为强势 |
| MA10 | 双周线 | 短线趋势辅助确认 |
| MA20 | 月线 | 中线生命线，机构常用的波段操作参考 |
| MA30 | — | 与MA20配合使用 |
| MA60 | 季线 | 中期趋势分水岭，是A股最重要的均线之一 |
| MA120 | 半年线 | 牛熊分界线（俗称"半年线定牛熊"） |
| MA250 | 年线 | 长期趋势基准，"年线之下不言底" |

#### 2.3 均线的实战用法

**（1）金叉与死叉**

- **金叉（Golden Cross）**：短期均线由下向上穿越长期均线，是买入信号。最常见且有效的是MA5上穿MA20、MA20上穿MA60。
- **死叉（Death Cross）**：短期均线由上向下穿越长期均线，是卖出信号。

金叉/死叉的可靠性取决于：
- 发生位置（低位金叉可靠性高于高位金叉）
- 均线斜率（均线向上倾斜的金叉更可靠）
- 成交量配合（金叉放量增强信号）

**（2）多头排列与空头排列**

```
多头排列（上涨趋势）：
  价格
   ↑    MA5 ─────────────  (最上方)
   │    MA20 ────────────
   │    MA60 ────────────
   │    MA120 ───────────
   │    MA250 ─────────── (最下方)
   └────────────────────→ 时间

空头排列（下跌趋势）：
  价格
   ↑    MA250 ──────────  (最上方)
   │    MA120 ───────────
   │    MA60  ───────────
   │    MA20  ───────────
   │    MA5   ─────────── (最下方)
   └────────────────────→ 时间
```

- **多头排列**：短期均线在长期均线上方，且所有均线均向上发散，是典型的牛市特征。
- **空头排列**：短期均线在长期均线下方，且所有均线均向下发散，是典型的熊市特征。

**（3）均线的支撑与压力**

当价格在均线上方运行时，均线（尤其是MA20和MA60）在回调时形成支撑；当价格在均线下方运行时，均线在反弹时形成压力。这一特性在震荡市中尤为显著——价格往往在触及关键均线后出现明确的方向性反应。

> **一句话总结：金叉买、死叉卖，多头排列持股不动，空头排列空仓观望——但世上没有100%准确的信号。**

#### 2.4 均线系统Python实战

**代码1：计算SMA和EMA**

```python
import pandas as pd
import numpy as np

def calculate_ma(df, price_col='close', periods=[5, 10, 20, 60, 120, 250]):
    """
    计算SMA和EMA

    参数:
        df: 包含价格数据的DataFrame
        price_col: 价格列名
        periods: 均线周期列表

    返回:
        添加了均线列的DataFrame
    """
    result = df.copy()

    for period in periods:
        # 简单移动平均
        result[f'SMA_{period}'] = result[price_col].rolling(window=period).mean()

        # 指数移动平均 (span=period 等价于 alpha=2/(period+1))
        result[f'EMA_{period}'] = result[price_col].ewm(span=period, adjust=False).mean()

    return result

# 示例：用模拟数据演示
np.random.seed(42)
dates = pd.date_range('2024-01-01', periods=120, freq='B')
np.random.seed(42)
price = 100 + np.cumsum(np.random.randn(120) * 1.5)
price = np.maximum(price, 30)  # 确保价格非负
sample_df = pd.DataFrame({'close': price}, index=dates)
sample_df['open'] = sample_df['close'] * (1 + np.random.randn(120) * 0.005)
sample_df['high'] = sample_df[['open', 'close']].max(axis=1) + np.abs(np.random.randn(120) * 0.8)
sample_df['low'] = sample_df[['open', 'close']].min(axis=1) - np.abs(np.random.randn(120) * 0.8)

result = calculate_ma(sample_df)
print("均线计算结果（最近5个交易日）：")
print(result[['close', 'SMA_5', 'EMA_5', 'SMA_20', 'EMA_20']].tail(5).round(2))
```

**代码2：检测金叉死叉**

```python
import pandas as pd
import numpy as np

def detect_cross(df, fast_col, slow_col):
    """
    检测两条均线的金叉和死叉

    金叉: 快线上穿慢线 (前一天快线<=慢线 且 当天快线>慢线)
    死叉: 快线下穿慢线 (前一天快线>=慢线 且 当天快线<慢线)

    参数:
        df: 包含均线数据的DataFrame
        fast_col: 快线（短期均线）列名
        slow_col: 慢线（长期均线）列名

    返回:
        添加了 cross_signal 列的DataFrame
            1 = 金叉
           -1 = 死叉
            0 = 无交叉
    """
    result = df.copy()

    # 计算差值
    diff = result[fast_col] - result[slow_col]
    diff_prev = diff.shift(1)

    # 金叉：前一日 <= 0 且当日 > 0
    golden_cross = (diff_prev <= 0) & (diff > 0)

    # 死叉：前一日 >= 0 且当日 < 0
    death_cross = (diff_prev >= 0) & (diff < 0)

    result['cross_signal'] = 0
    result.loc[golden_cross, 'cross_signal'] = 1
    result.loc[death_cross, 'cross_signal'] = -1

    return result

# 演示：检测MA5和MA20的金叉死叉
sample_with_ma = calculate_ma(sample_df, periods=[5, 20])
cross_result = detect_cross(sample_with_ma, 'SMA_5', 'SMA_20')

golden_dates = cross_result[cross_result['cross_signal'] == 1].index
death_dates = cross_result[cross_result['cross_signal'] == -1].index

print(f"MA5与MA20金叉次数: {len(golden_dates)}")
print(f"金叉日期: {golden_dates.tolist()}")
print(f"\nMA5与MA20死叉次数: {len(death_dates)}")
print(f"死叉日期: {death_dates.tolist()}")
```

**代码3：均线与价格可视化**

```python
import matplotlib.pyplot as plt
import matplotlib.dates as mdates

# 设置中文字体（如系统无中文字体，可替换为英文标签或安装SimHei）
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def plot_ma_with_cross(df, price_col='close', ma_periods=[5, 20, 60]):
    """
    绘制价格与均线，标注金叉死叉

    参数:
        df: 包含价格和均线数据的DataFrame
        price_col: 价格列名
        ma_periods: 要绘制的均线周期
    """
    fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 10),
                                    gridspec_kw={'height_ratios': [3, 1]})

    # 主图：价格与均线
    ax1.plot(df.index, df[price_col], color='black', linewidth=1.0, alpha=0.8, label='收盘价')
    colors = ['#FF6B35', '#004E89', '#1A936F', '#FFC857', '#C73E1D', '#6A4C93']
    for i, p in enumerate(ma_periods):
        if f'SMA_{p}' in df.columns:
            ax1.plot(df.index, df[f'SMA_{p}'], color=colors[i % len(colors)],
                     linewidth=0.8, alpha=0.7, label=f'MA{p}')

    # 标注金叉死叉
    if 'cross_signal' in df.columns:
        golden_cross_dates = df[df['cross_signal'] == 1].index
        death_cross_dates = df[df['cross_signal'] == -1].index
        for d in golden_cross_dates:
            if d in df.index:
                ax1.axvline(x=d, color='red', linestyle='--', alpha=0.3, linewidth=0.8)
        for d in death_cross_dates:
            if d in df.index:
                ax1.axvline(x=d, color='green', linestyle='--', alpha=0.3, linewidth=0.8)
        # 标注图例说明
        ax1.plot([], [], color='red', linestyle='--', alpha=0.5, label='金叉')
        ax1.plot([], [], color='green', linestyle='--', alpha=0.5, label='死叉')

    ax1.set_title('均线系统与交叉信号', fontsize=14, fontweight='bold')
    ax1.set_ylabel('价格', fontsize=12)
    ax1.legend(loc='upper left', fontsize=9, ncol=2)
    ax1.grid(True, alpha=0.3)

    # 副图：成交量（如有）
    if 'volume' in df.columns:
        ax2.bar(df.index, df['volume'], color='#7FB069', alpha=0.6, width=0.8)
        ax2.set_ylabel('成交量', fontsize=12)
    else:
        # 如无成交量数据，展示均线差值
        diff = df[f'SMA_{ma_periods[0]}'] - df[f'SMA_{ma_periods[1]}']
        ax2.fill_between(df.index, diff, 0,
                          where=(diff >= 0), color='red', alpha=0.3, label='快线在上')
        ax2.fill_between(df.index, diff, 0,
                          where=(diff < 0), color='green', alpha=0.3, label='快线在下')
        ax2.axhline(y=0, color='black', linewidth=0.5)
        ax2.set_ylabel('MA差值', fontsize=12)
        ax2.legend(loc='upper left', fontsize=9)

    ax2.set_xlabel('日期', fontsize=12)
    ax2.grid(True, alpha=0.3)

    # X轴日期格式
    ax1.xaxis.set_major_formatter(mdates.DateFormatter('%Y-%m'))
    plt.xticks(rotation=45)

    plt.tight_layout()
    plt.show()

# 执行可视化
sample_final = calculate_ma(sample_df, periods=[5, 20, 60])
sample_final = detect_cross(sample_final, 'SMA_5', 'SMA_20')
plot_ma_with_cross(sample_final, ma_periods=[5, 20, 60])
```

### 第三章 经典技术指标

#### 3.1 MACD——异同移动平均线

**（1）计算原理**

MACD由三个核心组件构成：

- **DIF（快线）**：EMA12 − EMA26，反映短期与长期趋势的差值。
- **DEA（慢线/信号线）**：DIF的EMA9，对DIF进行平滑处理。
- **MACD柱（Histogram）**：`(DIF − DEA) × 2`，直观展示DIF与DEA的分离程度。

A股默认参数（12, 26, 9）源自Gerald Appel的原始设计，但投资者可根据标的波动特性调整——高波动品种可适当增大参数以过滤噪音。

**（2）实战信号体系**

| 信号类型 | 条件 | 含义 |
|----------|------|------|
| MACD金叉 | DIF上穿DEA，MACD柱由负转正 | 短期看涨 |
| MACD死叉 | DIF下穿DEA，MACD柱由正转负 | 短期看跌 |
| 零轴上金叉 | DIF与DEA均在零轴上方金叉 | 多头市场中的回调买入机会 |
| 零轴下死叉 | DIF与DEA均在零轴下方死叉 | 空头市场中的反弹卖出信号 |
| 顶背离 | 价格创新高，但DIF/DEA未创新高 | 上涨动能衰减，警惕反转 |
| 底背离 | 价格创新低，但DIF/DEA未创新低 | 下跌动能衰减，关注反转 |

> **背离是MACD最有价值的信号，但背离不等于立即反转——背离之后可能还有二次背离。**

**（3）MACD Python完整实现（含背离检测）**

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def calculate_macd(df, price_col='close', fast=12, slow=26, signal=9):
    """
    计算MACD指标

    参数:
        df: 包含价格数据的DataFrame
        price_col: 价格列名
        fast: 快线周期（默认12）
        slow: 慢线周期（默认26）
        signal: 信号线周期（默认9）

    返回:
        添加了 DIF, DEA, MACD_hist 列的DataFrame
    """
    result = df.copy()

    # 计算快线和慢线的EMA
    ema_fast = result[price_col].ewm(span=fast, adjust=False).mean()
    ema_slow = result[price_col].ewm(span=slow, adjust=False).mean()

    # DIF = EMA12 - EMA26
    result['DIF'] = ema_fast - ema_slow

    # DEA = DIF的EMA9
    result['DEA'] = result['DIF'].ewm(span=signal, adjust=False).mean()

    # MACD柱 = (DIF - DEA) * 2
    result['MACD_hist'] = (result['DIF'] - result['DEA']) * 2

    # 金叉死叉
    diff = result['DIF'] - result['DEA']
    diff_prev = diff.shift(1)
    result['macd_cross'] = 0
    result.loc[(diff_prev <= 0) & (diff > 0), 'macd_cross'] = 1
    result.loc[(diff_prev >= 0) & (diff < 0), 'macd_cross'] = -1

    return result

def detect_macd_divergence(df, price_col='close', lookback=40, min_distance=10):
    """
    检测MACD顶背离和底背离

    顶背离: 价格创新高，但DIF未创新高
    底背离: 价格创新低，但DIF未创新低

    参数:
        df: 包含价格和MACD数据的DataFrame
        price_col: 价格列名
        lookback: 回溯窗口长度
        min_distance: 两个极值点之间的最小距离

    返回:
        添加了 divergence 列的DataFrame: 1=顶背离, -1=底背离, 0=无背离
    """
    result = df.copy()
    result['divergence'] = 0

    if len(result) < lookback:
        return result

    for i in range(lookback, len(result)):
        window = result.iloc[i - lookback:i + 1]
        current_idx = window.index[-1]
        current_price = window[price_col].iloc[-1]
        current_dif = window['DIF'].iloc[-1]

        # 寻找窗口内的价格和DIF极值
        price_max_idx = window[price_col].idxmax()
        price_min_idx = window[price_col].idxmin()
        dif_max = window['DIF'].max()
        dif_min = window['DIF'].min()

        # 顶背离：当前价格是窗口最高（或接近），但DIF不是最高
        if (current_price >= window[price_col].max() * 0.995 and
            current_dif < dif_max * 0.90 and
            current_dif > 0):
            peak_loc = window.index.get_loc(current_idx)
            dif_max_loc = window.index.get_loc(window['DIF'].idxmax())
            if abs(peak_loc - dif_max_loc) >= min_distance:
                result.loc[current_idx, 'divergence'] = 1

        # 底背离：当前价格是窗口最低（或接近），但DIF不是最低
        if (current_price <= window[price_col].min() * 1.005 and
            current_dif > dif_min * 1.10 and
            current_dif < 0):
            trough_loc = window.index.get_loc(current_idx)
            dif_min_loc = window.index.get_loc(window['DIF'].idxmin())
            if abs(trough_loc - dif_min_loc) >= min_distance:
                result.loc[current_idx, 'divergence'] = -1

    return result

def plot_macd_full(df, price_col='close'):
    """
    完整的MACD可视化：价格走势 + MACD指标 + 背离标注
    """
    fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(16, 10),
                                    gridspec_kw={'height_ratios': [2, 1]})

    # 上部分：价格走势
    ax1.plot(df.index, df[price_col], color='#1a1a2e', linewidth=1.2, label='收盘价')

    # 标注背离
    top_div = df[df['divergence'] == 1]
    bottom_div = df[df['divergence'] == -1]
    ax1.scatter(top_div.index, top_div[price_col], color='red', s=100,
                marker='v', zorder=5, label='顶背离')
    ax1.scatter(bottom_div.index, bottom_div[price_col], color='green', s=100,
                marker='^', zorder=5, label='底背离')

    ax1.set_title('MACD背离信号', fontsize=15, fontweight='bold')
    ax1.set_ylabel('价格', fontsize=12)
    ax1.legend(loc='upper left', fontsize=10)
    ax1.grid(True, alpha=0.3)

    # 下部分：MACD指标
    ax2.plot(df.index, df['DIF'], color='#E63946', linewidth=1.0, label='DIF')
    ax2.plot(df.index, df['DEA'], color='#457B9D', linewidth=1.0, label='DEA')

    # MACD柱：正值红色，负值绿色
    colors_hist = np.where(df['MACD_hist'] >= 0, '#E63946', '#2A9D8F')
    ax2.bar(df.index, df['MACD_hist'], color=colors_hist, width=0.8, alpha=0.5, label='MACD柱')

    ax2.axhline(y=0, color='black', linewidth=0.5, linestyle='-')
    ax2.set_ylabel('MACD', fontsize=12)
    ax2.set_xlabel('日期', fontsize=12)
    ax2.legend(loc='upper left', fontsize=10)
    ax2.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.show()
```

#### 3.2 RSI——相对强弱指标

**（1）计算公式**

RSI由Welles Wilder于1978年提出，核心思想是衡量一定时期内价格上涨幅度占总波动幅度的比例：

```
RS = N日内上涨收盘价的平均涨幅 / N日内下跌收盘价的平均跌幅（取绝对值）
RSI = 100 − 100 / (1 + RS)
```

标准的14日RSI计算步骤：
1. 计算每日涨跌幅度（当日收盘 − 前日收盘）。
2. 将14日内的涨幅取平均（AU），跌幅取绝对值后取平均（AD）。
3. `RS = AU / AD`，代入公式得RSI值，范围始终在0到100之间。

**（2）经典信号**

| RSI区间 | 状态 | 操作参考 |
|----------|------|----------|
| > 70 | 超买 | 警惕回调，但强势趋势中可长期停留在超买区 |
| < 30 | 超卖 | 关注反弹，但单边下跌中可长期停留在超卖区 |
| 50附近 | 中性 | 无明显方向性信号 |

**钝化现象**：在强劲的单边行情中，RSI可以长时间停留在超买（>70）或超卖（<30）区域而不发生反转。此时若机械地"超买卖出、超卖买入"将招致重大亏损。钝化本身也是一种信号——它表明当前趋势极强，不宜逆势交易。

**（3）RSI Python实现**

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def calculate_rsi(df, price_col='close', period=14):
    """
    计算RSI指标

    参数:
        df: 包含价格数据的DataFrame
        price_col: 价格列名
        period: RSI周期（默认14）

    返回:
        添加了 RSI 列的DataFrame
    """
    result = df.copy()

    # 计算每日价格变动
    delta = result[price_col].diff()

    # 分别计算涨幅和跌幅
    gain = delta.where(delta > 0, 0.0)
    loss = (-delta).where(delta < 0, 0.0)

    # 使用Wilder平滑法（与EMA的alpha不同，Wilder使用 1/period）
    avg_gain = gain.ewm(alpha=1/period, adjust=False).mean()
    avg_loss = loss.ewm(alpha=1/period, adjust=False).mean()

    # RS = 平均涨幅 / 平均跌幅
    rs = avg_gain / avg_loss

    # RSI = 100 - 100 / (1 + RS)
    result['RSI'] = 100.0 - (100.0 / (1.0 + rs))

    # 信号列
    result['rsi_signal'] = '中性'
    result.loc[result['RSI'] > 70, 'rsi_signal'] = '超买'
    result.loc[result['RSI'] < 30, 'rsi_signal'] = '超卖'

    return result

def plot_rsi(df, price_col='close'):
    """
    绘制价格与RSI双图
    """
    fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 9),
                                    gridspec_kw={'height_ratios': [2, 1]})

    # 价格走势
    ax1.plot(df.index, df[price_col], color='#1a1a2e', linewidth=1.0, label='收盘价')
    ax1.set_title('RSI相对强弱指标', fontsize=14, fontweight='bold')
    ax1.set_ylabel('价格', fontsize=12)
    ax1.legend(loc='upper left', fontsize=10)
    ax1.grid(True, alpha=0.3)

    # RSI
    ax2.plot(df.index, df['RSI'], color='#E76F51', linewidth=1.0, label='RSI(14)')
    ax2.axhline(y=70, color='red', linestyle='--', linewidth=0.8, alpha=0.7, label='超买线(70)')
    ax2.axhline(y=30, color='green', linestyle='--', linewidth=0.8, alpha=0.7, label='超卖线(30)')
    ax2.axhline(y=50, color='gray', linestyle=':', linewidth=0.5, alpha=0.5)
    ax2.fill_between(df.index, 70, 100, alpha=0.1, color='red')
    ax2.fill_between(df.index, 0, 30, alpha=0.1, color='green')
    ax2.set_ylim(0, 100)
    ax2.set_ylabel('RSI', fontsize=12)
    ax2.set_xlabel('日期', fontsize=12)
    ax2.legend(loc='upper left', fontsize=10)
    ax2.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.show()
```

#### 3.3 布林带（Bollinger Bands）

**（1）构成原理**

布林带由John Bollinger于1980年代提出，由三条轨道构成：

- **中轨（Middle Band）**：N日SMA（通常N=20）
- **上轨（Upper Band）**：中轨 + K × N日标准差（通常K=2）
- **下轨（Lower Band）**：中轨 − K × N日标准差（通常K=2）

在正态分布假设下，约95%的价格数据应落在上轨与下轨之间。当价格触及或突破轨道边界时，表明价格处于统计意义上的极端位置。

**（2）核心信号**

| 信号 | 表现 | 含义 |
|------|------|------|
| 带宽收缩 | 上轨与下轨间距收窄 | 市场低波动，即将选择方向（"布林带收口"） |
| 带宽扩张 | 上轨与下轨间距扩大 | 市场高波动，趋势可能加速 |
| 触及上轨 | 价格接近或突破上轨 | 强势特征，但短期可能超买 |
| 触及下轨 | 价格接近或突破下轨 | 弱势特征，但短期可能超卖 |
| 沿轨道运行 | 价格持续贴着上轨/下轨 | 强劲趋势中常见，不宜逆势 |

**（3）布林带Python实现**

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def calculate_bollinger(df, price_col='close', period=20, num_std=2):
    """
    计算布林带

    参数:
        df: 包含价格数据的DataFrame
        price_col: 价格列名
        period: 中轨周期（默认20）
        num_std: 标准差倍数（默认2）

    返回:
        添加了 boll_mid, boll_upper, boll_lower, boll_width, boll_pct_b 列的DataFrame
    """
    result = df.copy()

    # 中轨 = N日SMA
    result['boll_mid'] = result[price_col].rolling(window=period).mean()

    # 标准差
    rolling_std = result[price_col].rolling(window=period).std()

    # 上轨和下轨
    result['boll_upper'] = result['boll_mid'] + num_std * rolling_std
    result['boll_lower'] = result['boll_mid'] - num_std * rolling_std

    # 带宽 = (上轨 - 下轨) / 中轨 — 衡量相对波动率
    result['boll_width'] = (result['boll_upper'] - result['boll_lower']) / result['boll_mid']

    # %b = (价格 - 下轨) / (上轨 - 下轨) — 价格在布林带中的相对位置
    band_range = result['boll_upper'] - result['boll_lower']
    result['boll_pct_b'] = (result[price_col] - result['boll_lower']) / band_range

    return result

def plot_bollinger(df, price_col='close'):
    """
    绘制布林带完整图表
    """
    fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 9),
                                    gridspec_kw={'height_ratios': [3, 1]})

    # 主图：价格与布林带
    ax1.plot(df.index, df[price_col], color='#1a1a2e', linewidth=1.0, label='收盘价')
    ax1.plot(df.index, df['boll_upper'], color='#E63946', linewidth=0.8,
             linestyle='--', alpha=0.7, label='上轨')
    ax1.plot(df.index, df['boll_mid'], color='#457B9D', linewidth=0.8,
             linestyle='-', alpha=0.6, label='中轨(MA20)')
    ax1.plot(df.index, df['boll_lower'], color='#2A9D8F', linewidth=0.8,
             linestyle='--', alpha=0.7, label='下轨')
    ax1.fill_between(df.index, df['boll_upper'], df['boll_lower'],
                      alpha=0.08, color='#457B9D')
    ax1.set_title('布林带 (20, 2)', fontsize=14, fontweight='bold')
    ax1.set_ylabel('价格', fontsize=12)
    ax1.legend(loc='upper left', fontsize=10)
    ax1.grid(True, alpha=0.3)

    # 副图：带宽
    ax2.plot(df.index, df['boll_width'], color='#E76F51', linewidth=1.0, label='带宽')
    ax2.axhline(y=df['boll_width'].mean(), color='gray', linestyle=':',
                linewidth=0.8, label=f'均值 ({df["boll_width"].mean():.3f})')
    ax2.set_ylabel('带宽', fontsize=12)
    ax2.set_xlabel('日期', fontsize=12)
    ax2.legend(loc='upper left', fontsize=10)
    ax2.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.show()

# 演示

np.random.seed(789)
```

#### 3.4 KDJ——随机指标

KDJ由George Lane提出，通过比较当前价格与一定时期内的价格区间来判断超买超卖。

**计算步骤（以9日KDJ为例）：**

1. `RSV = (当日收盘 − 9日最低) / (9日最高 − 9日最低) × 100`
2. `K = (2/3) × 前一日K + (1/3) × 当日RSV`
3. `D = (2/3) × 前一日D + (1/3) × 当日K`
4. `J = 3 × K − 2 × D`

**信号体系：**

- K值、D值在80以上为超买区，20以下为超卖区。
- K线上穿D线为金叉，下穿为死叉。
- J值大于100为严重超买，小于0为严重超卖。
- KDJ与价格的顶/底背离信号类似MACD。

**KDJ的局限性**：KDJ波动极为频繁，在单边行情中钝化严重，假信号远多于MACD和RSI。实战中KDJ更适合作为辅助确认指标，而非独立决策依据。

> **一句话总结：KDJ反应最快但假信号最多，适用于震荡市中的短线波段，单边市中请关闭KDJ。**

#### 3.5 成交量分析

成交量是技术分析中唯一不可操纵（或操纵成本极高）的先行指标，A股市场尤其重视量价关系。

**（1）量价配合的四种基本模式：**

| 模式 | 价格 | 成交量 | 含义 |
|------|------|--------|------|
| 量增价升 | 上涨 | 放大 | 上涨健康，趋势确认 |
| 量缩价升 | 上涨 | 萎缩 | 上涨乏力，警惕背离 |
| 量增价跌 | 下跌 | 放大 | 抛压沉重，下跌可能加速 |
| 量缩价跌 | 下跌 | 萎缩 | 抛压减轻，接近底部信号 |

**（2）重要量价谚语：**

- **"天量见天价"**：当成交量创出历史天量而价格处于高位时，往往是阶段性顶部。全市场（或个股）所有愿意买入的资金已经入场，后续买盘枯竭，价格面临回调压力。
- **"地量见地价"**：当成交量萎缩至极低水平时，说明卖方力量显著衰减，市场情绪低迷至极，往往是阶段性底部区域。但"地量之后还有地量"，需要结合其他信号确认。
- **"量在价先"**：成交量的变化往往领先于价格变化。价格突破前若成交量率先放大，突破的有效性更高。

**（3）换手率：**

换手率 = 当日成交量 / 流通股本 × 100%

| 换手率区间 | 参考含义 |
|------------|----------|
| < 1% | 交投清淡，关注度低 |
| 1% - 3% | 正常交投水平 |
| 3% - 7% | 交易活跃，关注度高 |
| 7% - 15% | 高度活跃，可能有资金进出 |
| > 15% | 极度活跃，注意主力出货或建仓 |

A股中小盘股换手率普遍高于大盘蓝筹。分析换手率时需结合个股历史水平和市值规模综合判断。

> **一句话总结：价格可能骗人，成交量不会——放量突破是真突破，缩量上涨是纸老虎。**

#### 4.3 模拟策略回测Python代码

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def simulate_ma_cross_strategy(df, fast_period=5, slow_period=20, initial_capital=100000):
    """
    基于均线金叉死叉的模拟策略回测

    策略规则:
      - 金叉时全仓买入
      - 死叉时全部卖出
      - 忽略交易成本和滑点

    参数:
        df: 包含价格数据的DataFrame (需含'close'列)
        fast_period: 快线周期
        slow_period: 慢线周期
        initial_capital: 初始资金

    返回:
        策略回测结果DataFrame
    """
    result = df.copy()

    # 计算均线
    result['fast_ma'] = result['close'].rolling(window=fast_period).mean()
    result['slow_ma'] = result['close'].rolling(window=slow_period).mean()

    # 检测交叉信号
    diff = result['fast_ma'] - result['slow_ma']
    diff_prev = diff.shift(1)
    result['signal'] = 0
    result.loc[(diff_prev <= 0) & (diff > 0), 'signal'] = 1   # 金叉买入
    result.loc[(diff_prev >= 0) & (diff < 0), 'signal'] = -1  # 死叉卖出

    # 计算持仓状态（1=持仓, 0=空仓）
    result['position'] = 0
    in_position = False
    for i in range(len(result)):
        if result['signal'].iloc[i] == 1:
            in_position = True
        elif result['signal'].iloc[i] == -1:
            in_position = False
        result.iloc[i, result.columns.get_loc('position')] = 1 if in_position else 0

    # 计算每日收益率
    result['daily_return'] = result['close'].pct_change()

    # 策略收益 = 持仓状态 * 日收益率
    result['strategy_return'] = result['position'].shift(1) * result['daily_return']
    result['strategy_return'] = result['strategy_return'].fillna(0)

    # 累积收益
    result['cum_market'] = (1 + result['daily_return'].fillna(0)).cumprod()
    result['cum_strategy'] = (1 + result['strategy_return']).cumprod()

    # 计算策略指标
    total_return = result['cum_strategy'].iloc[-1] - 1
    market_return = result['cum_market'].iloc[-1] - 1

    # 年化收益率（假设252个交易日）
    years = len(result) / 252
    annual_return = (1 + total_return) ** (1 / years) - 1 if years > 0 else 0
    market_annual = (1 + market_return) ** (1 / years) - 1 if years > 0 else 0

    # 最大回撤
    cum_max = result['cum_strategy'].cummax()
    drawdown = (result['cum_strategy'] - cum_max) / cum_max
    max_drawdown = drawdown.min()

    # 交易次数
    trades = (result['signal'].abs() == 1).sum()

    # 胜率（卖出时的盈亏判断）
    sell_signals = result[result['signal'] == -1].index
    buy_signals = result[result['signal'] == 1].index
    wins = 0
    total_trades_pair = 0
    if len(buy_signals) > 0 and len(sell_signals) > 0:
        for sell_idx in sell_signals:
            prev_buys = buy_signals[buy_signals < sell_idx]
            if len(prev_buys) > 0:
                last_buy_idx = prev_buys[-1]
                buy_price = result.loc[last_buy_idx, 'close']
                sell_price = result.loc[sell_idx, 'close']
                if sell_price > buy_price:
                    wins += 1
                total_trades_pair += 1

    win_rate = wins / total_trades_pair if total_trades_pair > 0 else 0

    print("=" * 50)
    print(f"均线策略回测 (MA{fast_period} / MA{slow_period})")
    print("=" * 50)
    print(f"策略总收益率: {total_return*100:.2f}%")
    print(f"买入持有收益率: {market_return*100:.2f}%")
    print(f"策略年化收益率: {annual_return*100:.2f}%")
    print(f"买入持有年化收益率: {market_annual*100:.2f}%")
    print(f"最大回撤: {max_drawdown*100:.2f}%")
    print(f"交易次数: {trades}")
    print(f"完整交易对数: {total_trades_pair}")
    print(f"胜率: {win_rate*100:.1f}%")
    print("=" * 50)

    return result

def plot_strategy_result(result):
    """
    绘制策略回测对比图
    """
    fig, (ax1, ax2, ax3) = plt.subplots(3, 1, figsize=(14, 11),
                                         gridspec_kw={'height_ratios': [3, 1, 1]})

    # 图1：价格、均线、买卖信号
    ax1.plot(result.index, result['close'], color='black', linewidth=0.8, alpha=0.8, label='收盘价')
    ax1.plot(result.index, result['fast_ma'], color='#E63946', linewidth=0.8,
             alpha=0.7, label=f'MA{5}')
    ax1.plot(result.index, result['slow_ma'], color='#457B9D', linewidth=0.8,
             alpha=0.7, label=f'MA{20}')

    buy_points = result[result['signal'] == 1]
    sell_points = result[result['signal'] == -1]
    ax1.scatter(buy_points.index, buy_points['close'], color='red', marker='^',
                s=60, zorder=5, label='金叉买入')
    ax1.scatter(sell_points.index, sell_points['close'], color='green', marker='v',
                s=60, zorder=5, label='死叉卖出')

    ax1.set_title('均线交叉策略回测', fontsize=14, fontweight='bold')
    ax1.set_ylabel('价格', fontsize=12)
    ax1.legend(loc='upper left', fontsize=9, ncol=2)
    ax1.grid(True, alpha=0.3)

    # 图2：累积收益对比
    ax2.plot(result.index, result['cum_strategy'], color='#E63946', linewidth=1.2,
             label='策略累积收益')
    ax2.plot(result.index, result['cum_market'], color='#457B9D', linewidth=1.0,
             label='买入持有累积收益')
    ax2.axhline(y=1, color='black', linewidth=0.5, linestyle=':')
    ax2.set_ylabel('累积收益倍数', fontsize=12)
    ax2.legend(loc='upper left', fontsize=10)
    ax2.grid(True, alpha=0.3)

    # 图3：回撤曲线
    cum_max = result['cum_strategy'].cummax()
    drawdown = (result['cum_strategy'] - cum_max) / cum_max
    ax3.fill_between(result.index, 0, drawdown, color='#E63946', alpha=0.3)
    ax3.plot(result.index, drawdown, color='#E63946', linewidth=0.8)
    ax3.set_ylabel('回撤', fontsize=12)
    ax3.set_xlabel('日期', fontsize=12)
    ax3.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.show()

# 使用模拟的茅台数据运行回测
np.random.seed(42)
dates_mt = pd.date_range('2020-01-01', '2024-12-31', freq='W-FRI')
n_weeks = len(dates_mt)

# 模拟茅台走势：2020-2021上涨，2021-2024调整，2024反弹
trend1 = np.linspace(1000, 2600, 60)       # 2020.1 - 2021.3 上涨
trend2 = np.linspace(2600, 1300, 150)      # 2021.3 - 2024.1 调整
trend3 = np.linspace(1300, 1800, n_weeks - 210)  # 2024.1 - 2024.12 反弹
trend = np.concatenate([trend1, trend2, trend3])
trend = trend[:n_weeks]

np.random.seed(42)
noise_mt = np.cumsum(np.random.randn(n_weeks) * 40)
price_mt = trend + noise_mt
price_mt = np.maximum(price_mt, 800)

df_mt = pd.DataFrame({'close': price_mt}, index=dates_mt)

strategy_result = simulate_ma_cross_strategy(df_mt, fast_period=5, slow_period=20)
plot_strategy_result(strategy_result)
```

#### 4.4 技术分析的真实局限

在对茅台2020-2024年的技术信号进行全面复盘后，我们必须坦诚地讨论以下局限：

1. **假信号不可避免**：在上述区间内，MA5/MA20交叉策略共产生约25次交易信号，其中约40%为假信号（交叉后价格未朝预期方向运行）。在2021-2023年的震荡下跌市中，金叉买入后继续下跌的情况频繁发生。

2. **滞后性**：均线和MACD等趋势指标本质上都是滞后指标——它们在趋势确立后才发出信号。2020年3月的底背离信号确认时，价格已从最低点反弹约8%。

3. **参数敏感性**：将快线周期从5改为10或3，策略收益率可能发生显著变化——参数优化存在严重的过拟合风险。历史最优参数不等于未来最优参数。

4. **无法预测幅度**：技术信号能提示方向（涨/跌），但无法预测幅度。金叉之后是涨3%还是30%，技术分析无言以对。

5. **黑天鹅事件免疫**：任何技术指标都无法预测2021年的"双减"政策对教育股的冲击，或2024年1月的市场流动性危机——这些事件由基本面因素驱动。

> **技术分析的最大价值不是"预测未来"，而是提供一套纪律化的决策框架，帮助投资者在不确定性中保持一致的行动准则。**

### 第五章 三大财务报表速读

#### 5.1 利润表——企业的成绩单

利润表反映企业在一定时期内的经营成果，是投资者最关注的报表。要读懂利润表，需从顶层营业收入逐层穿透至底层净利润。

**利润表核心结构：**

```
营业收入（主营业务收入 + 其他业务收入）
  − 营业成本
──────────────────────────
  = 毛利润 → 毛利率 = 毛利润 / 营业收入
  − 销售费用
  − 管理费用
  − 研发费用
  − 财务费用
  + 其他收益 / 投资收益
──────────────────────────
  = 营业利润 → 营业利润率 = 营业利润 / 营业收入
  + 营业外收入 − 营业外支出
──────────────────────────
  = 利润总额
  − 所得税
──────────────────────────
  = 净利润 → 净利率 = 净利润 / 营业收入
```

**关键分析指标：**

| 指标 | 公式 | 含义 |
|------|------|------|
| 毛利率 | (营收 − 营业成本) / 营收 | 产品的定价权与成本控制能力 |
| 净利率 | 净利润 / 营收 | 综合盈利能力 |
| 同比增速 | (本期 − 去年同期) / 去年同期 | 趋势性增长判断 |
| 环比增速 | (本期 − 上期) / 上期 | 短期动能变化 |

以贵州茅台2023年年报为例：毛利率长期维持在91%以上，这一水平在A股中几乎无可匹敌，反映了茅台极强的品牌定价权和渠道控制力。相比之下，宁德时代2023年毛利率约22%，属于制造业正常水平——低毛利并非劣势，它与高周转率相结合同样可以创造优秀的ROE。

> **一句话总结：毛利率看商业模式（白酒>消费>制造），净利率看管理效率，同比看趋势，环比看动量。**

#### 5.2 资产负债表——企业的体检报告

资产负债表的恒等式：**资产 = 负债 + 所有者权益**

**（1）资产端解读**

```
流动资产（一年内可变现）          非流动资产（长期持有）
├── 货币资金                      ├── 固定资产（厂房、设备）
├── 应收账款（警惕高额挂账）       ├── 在建工程
├── 存货（警惕减值风险）           ├── 无形资产
├── 交易性金融资产                 ├── 商誉（并购溢价，减值风险大）
└── 预付款项                      └── 长期股权投资
```

**（2）负债端解读**

```
流动负债（一年内到期）            非流动负债（一年以上）
├── 短期借款                      ├── 长期借款
├── 应付账款（占用上游资金）       ├── 应付债券
├── 预收款项/合同负债              └── 递延所得税负债
└── 应付职工薪酬
```

**（3）关键风险指标**

| 指标 | 公式 | 警戒线 | 含义 |
|------|------|--------|------|
| 资产负债率 | 总负债 / 总资产 | > 70% 高度警惕 | 财务杠杆水平 |
| 流动比率 | 流动资产 / 流动负债 | < 1 流动性风险 | 短期偿债能力 |
| 商誉占比 | 商誉 / 净资产 | > 30% 警惕减值 | 并购溢价风险 |

**商誉风险案例**：A股历史上多家公司因大规模并购积累了巨额商誉，随后在业绩承诺期满后集中计提商誉减值，导致年度巨亏。投资者面对高商誉公司时，应重点审视被并购资产的盈利能力和行业前景。

> **一句话总结：资产负债表是企业的骨骼——杠杆过高易骨折，商誉过高易"暴雷"。**

#### 5.3 现金流量表——企业的血液

现金流量表分为三大板块：

**（1）经营活动现金流**：来自主营业务销售和采购的现金净流入，是企业最核心、最可持续的现金来源。持续为负的经营活动现金流是重大危险信号。

**（2）投资活动现金流**：来自购买/出售固定资产、对外投资等。处于快速扩张期的企业投资现金流通常为负（大量资本开支），这本身是中性的，关键在于投资效率和回报前景。

**（3）筹资活动现金流**：来自股权融资、银行贷款、债券发行和分红/回购。持续依赖筹资活动维持运营的公司存在流动性隐患。

**净利润含金量 = 经营活动现金流净额 / 净利润**

该比值大于1说明利润是"真金白银"，小于1尤其是持续显著小于1，说明利润的质量存疑——利润可能大量沉淀在应收账款和存货中。

> **一句话总结：利润可以是"做"出来的，现金流才是"挣"出来的——净利润含金量是识别财务造假的照妖镜。**

**第五章自测题：**

### 第六章 估值方法实战

#### 6.1 相对估值法

相对估值法通过寻找可比公司或对标历史估值水平，判断当前估值的相对高低。

**（1）市盈率（PE）**

`PE = 股价 / 每股收益 = 总市值 / 净利润`

PE是最常用的估值指标，代表市场愿意为每一元利润支付的价格。PE越低理论上越便宜，但低PE可能隐含增长前景不佳的预期。

**PE的两种使用方法：**

- **行业横向对比**：将标的PE与同行业可比公司比较。茅台PE为30倍而白酒行业平均为25倍，说明茅台享有品牌溢价——这可能是合理的，也可能是高估的。
- **历史分位数分析**：计算当前PE在标的自身历史PE分布中的分位。若当前PE处于过去5年的20%分位以下，则从历史角度看相对便宜；处于80%分位以上则相对昂贵。

**（2）市净率（PB）**

`PB = 股价 / 每股净资产 = 总市值 / 净资产`

PB适用于资产密集型行业（银行、保险、地产、钢铁），不适合轻资产企业（互联网、服务、品牌消费品）。银行股PB长期低于1（"破净"）在A股并不罕见，这不意味着银行股普遍低估——需要结合不良贷款率、ROE等因素综合判断。

**（3）市销率（PS）**

`PS = 总市值 / 营业收入`

PS适用于尚未盈利的高成长企业（如早期科技公司、创新药企）。这类企业利润为负，PE和PB均失效，PS可提供一个相对估值基准。但PS忽略了成本结构和盈利能力的差异。

#### 6.2 绝对估值法：DCF简化演示

**DCF（现金流折现模型）**的核心理念：企业的内在价值等于其未来所有自由现金流的现值之和。虽然完整的自由现金流折现模型需要详尽的财务预测，但其思想可以用简化版演示。

**简化DCF Python代码：**

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

def simple_dcf(
    initial_fcf,        # 首年自由现金流（亿元）
    growth_rate,        # 前N年增长率
    terminal_growth,    # 永续增长率
    wacc,               # 加权平均资本成本（折现率）
    high_growth_years=10,  # 高增长年限
    shares_outstanding=10   # 总股本（亿股）
):
    """
    简化版DCF估值模型

    模型结构：
        阶段1: high_growth_years年，以growth_rate增长
        阶段2: 永续增长期，以terminal_growth增长

    参数:
        initial_fcf: 首年自由现金流
        growth_rate: 高增长阶段增长率
        terminal_growth: 永续增长率（通常2%-4%，不超过GDP增速）
        wacc: 折现率（通常8%-12%）
        high_growth_years: 高增长年数
        shares_outstanding: 总股本（亿股）

    返回:
        每股内在价值估计
    """
    # 阶段1：逐年折现高增长期现金流
    pv_high_growth = 0
    fcf = initial_fcf
    cash_flows = []

    for year in range(1, high_growth_years + 1):
        fcf = fcf * (1 + growth_rate)
        pv = fcf / ((1 + wacc) ** year)
        pv_high_growth += pv
        cash_flows.append({'年份': year, '现金流': round(fcf, 2), '现值': round(pv, 2)})

    # 阶段2：永续增长终值（戈登增长模型）
    terminal_fcf = fcf * (1 + terminal_growth)
    terminal_value = terminal_fcf / (wacc - terminal_growth)
    pv_terminal = terminal_value / ((1 + wacc) ** high_growth_years)

    # 企业总价值
    enterprise_value = pv_high_growth + pv_terminal

    # 每股价值
    per_share_value = enterprise_value / shares_outstanding

    # 输出结果
    print("=" * 55)
    print("         简化DCF估值模型结果")
    print("=" * 55)
    print(f"  首年自由现金流:  {initial_fcf:.1f} 亿元")
    print(f"  高增长阶段增速:  {growth_rate*100:.1f}% (持续{high_growth_years}年)")
    print(f"  永续增长率:      {terminal_growth*100:.1f}%")
    print(f"  折现率(WACC):    {wacc*100:.1f}%")
    print(f"  总股本:          {shares_outstanding:.1f} 亿股")
    print("-" * 55)
    print(f"  高增长期现值:    {pv_high_growth:.1f} 亿元")
    print(f"  永续终值现值:    {pv_terminal:.1f} 亿元")
    print(f"  企业总价值:      {enterprise_value:.1f} 亿元")
    print(f"  每股内在价值:    {per_share_value:.2f} 元")
    print("=" * 55)

    return {
        'enterprise_value': enterprise_value,
        'per_share_value': per_share_value,
        'cash_flows': cash_flows,
        'pv_terminal': pv_terminal
    }

def dcf_sensitivity(initial_fcf, growth_rates, wacc_rates,
                    terminal_growth=0.03, high_growth_years=10,
                    shares_outstanding=10):
    """
    DCF参数敏感性分析：在不同增长率和折现率假设下的每股价值
    """
    results = np.zeros((len(growth_rates), len(wacc_rates)))

    for i, g in enumerate(growth_rates):
        for j, w in enumerate(wacc_rates):
            result = simple_dcf(
                initial_fcf=initial_fcf,
                growth_rate=g,
                terminal_growth=terminal_growth,
                wacc=w,
                high_growth_years=high_growth_years,
                shares_outstanding=shares_outstanding
            )
            results[i, j] = result['per_share_value']

    # 可视化敏感性热力图
    fig, ax = plt.subplots(figsize=(10, 7))
    im = ax.imshow(results, cmap='RdYlGn', aspect='auto')

    ax.set_xticks(range(len(wacc_rates)))
    ax.set_xticklabels([f'{w*100:.0f}%' for w in wacc_rates])
    ax.set_yticks(range(len(growth_rates)))
    ax.set_yticklabels([f'{g*100:.0f}%' for g in growth_rates])

    # 在单元格中标注数值
    for i in range(len(growth_rates)):
        for j in range(len(wacc_rates)):
            text_color = 'white' if results[i, j] < np.median(results) else 'black'
            ax.text(j, i, f'{results[i, j]:.1f}',
                    ha='center', va='center', color=text_color, fontsize=9)

    ax.set_xlabel('折现率 (WACC)', fontsize=12)
    ax.set_ylabel('高增长阶段增速', fontsize=12)
    ax.set_title('DCF每股价值敏感性分析（单位：元）', fontsize=14, fontweight='bold')
    plt.colorbar(im, ax=ax, label='每股价值（元）')
    plt.tight_layout()
    plt.show()

    return results

# 以某消费龙头企业为假设参数进行演示
print("\n>>> 假设参数（非真实预测，仅供方法演示）<<<\n")

result = simple_dcf(
    initial_fcf=60.0,        # 假设首年自由现金流60亿元
    growth_rate=0.12,        # 高增长期12%增长率
    terminal_growth=0.03,    # 永续增长3%
    wacc=0.09,               # 折现率9%
    high_growth_years=10,
    shares_outstanding=12.56  # 假设总股本12.56亿股
)

# 敏感性分析
print("\n>>> DCF参数敏感性分析 <<<\n")
growth_rates = [0.06, 0.08, 0.10, 0.12, 0.15, 0.18]
wacc_rates = [0.07, 0.08, 0.09, 0.10, 0.11, 0.12]
sensitivity_results = dcf_sensitivity(
    initial_fcf=60.0,
    growth_rates=growth_rates,
    wacc_rates=wacc_rates,
    terminal_growth=0.03,
    high_growth_years=10,
    shares_outstanding=12.56
)
```

**DCF的局限性必须明确指出**：
1. 终值（Terminal Value）通常占企业总价值的60%-80%，而终值对永续增长率和折现率的假设极度敏感——微小的参数变化可导致估值结果翻倍或腰斩。
2. 未来十年以上的现金流预测本质上是一种"有根据的猜测"。
3. 实战中，DCF更适合用来理解企业价值驱动因素（你需要多高的增长率才能"支撑"当前股价？），而非作为精确的买卖价格依据。

> **一句话总结：DCF告诉你"这个价格隐含了市场对增长率的什么预期"，而不是告诉你"这只股票值多少钱"。**

#### 6.3 PEG——成长性修正的市盈率

`PEG = PE / 盈利增长率（%）`

PEG由传奇基金经理彼得·林奇（Peter Lynch）推广，核心逻辑是：高增长的股票理应享有高PE，关键在于PE与增长率是否匹配。

| PEG值 | 参考解读 |
|-------|----------|
| < 0.5 | 可能严重低估（或市场预期增长将大幅放缓） |
| 0.5 - 1.0 | 估值相对合理至偏低 |
| 1.0 - 2.0 | 估值相对合理至偏高 |
| > 2.0 | 估值偏高（或市场预期增长将加速） |

使用PEG时需要注意：
- 盈利增长率应使用未来3-5年的预期复合增长率，而非过去的历史数据。
- 周期股（钢铁、化工、航运）在盈利高峰期PE极低，但此时计算出的PEG会误导性地显示"低估"——因为未来盈利可能大幅下滑。
- PEG适合稳定成长型公司（消费、医药、科技），不适合周期股和亏损公司。

> **一句话总结：PEG是PE的"成长性校准器"——30倍PE配30%增长（PEG=1）比10倍PE配2%增长（PEG=5）更合理。**

### 第七章 行业与宏观分析

#### 7.1 行业生命周期

任何行业都会经历从诞生到衰落的过程，不同阶段的投资逻辑截然不同：

```
行业规模
  ↑
  │         ┌──────────┐
  │         │   成熟期   │  ┌──────┐
  │         │  (稳定增长) │  │衰退期 │
  │    ┌────┤           ├──┤      │
  │    │成长期           │  │      │
  │   ┌┤  (快速增长)     │  └──────┘
  │  ┌┤                 │
  │ ┌┤                  │
  │┌┤导入期              │
  └┴┴──────────────────────────→ 时间
```

| 阶段 | 特征 | 投资策略 | A股案例 |
|------|------|----------|---------|
| 导入期 | 技术不成熟，市场认知度低，亏损普遍 | 高风险高回报，适合风投 | 固态电池（2024年） |
| 成长期 | 市场快速扩大，竞争格局未定，龙头初现 | 寻找赛道龙头，享受戴维斯双击 | 新能源汽车（2020-2022年） |
| 成熟期 | 增长放缓，格局稳固，龙头盈利丰厚 | 投资龙头，看重分红和确定性 | 白酒行业 |
| 衰退期 | 需求萎缩，产能过剩，价格战 | 规避或寻找转型机会 | 传统燃油车零部件 |

**宁德时代案例分析**：动力电池行业在2019-2022年处于成长期中段，宁德时代凭借技术优势和规模效应，市场份额从约40%提升至50%以上。进入2023-2024年，行业增速放缓、二线电池厂产能释放、下游车企自研电池，行业逐渐步入成长后期向成熟期过渡——竞争加剧、毛利率承压成为2024年的核心矛盾。

#### 7.2 波特五力简化版

迈克尔·波特的五力模型是分析行业竞争格局的经典框架：

| 竞争力量 | 核心问题 | 行业利润率的"杀手" |
|----------|----------|---------------------|
| 现有企业竞争 | 行业内有多少竞争者？竞争有多激烈？ | 价格战、广告战、研发竞赛 |
| 新进入者威胁 | 进入壁垒高吗？（资金、技术、牌照） | 产能过剩、拉低价格 |
| 替代品威胁 | 客户能否轻易转向替代品？ | 需求萎缩、定价权丧失 |
| 供应商议价力 | 供应商是否集中？能否轻易更换？ | 成本上升、利润被挤占 |
| 买方议价力 | 客户是否集中？转换成本高吗？ | 价格承压、回款变慢 |

**应用于A股行业判断：**
- 白酒行业：五力中最强的是品牌壁垒（新进入者极难突破），最弱的是买方议价力（消费者分散）。这是白酒行业长期高毛利率的根源。
- 光伏行业（2023-2024年）：五力全面恶化——现有竞争白热化（产能严重过剩）、新进入者不断（跨界资本涌入）、替代品风险（钙钛矿等新技术）、买方议价力强（下游电站集中采购）、供应商议价力有限。这是光伏企业在2024年普遍亏损的结构性原因。

#### 7.3 宏观经济指标速览

| 指标 | 含义 | 对股市的典型影响 |
|------|------|------------------|
| GDP增速 | 经济总产出增长率 | GDP增速上行利好周期性行业（钢铁、化工、有色）；下行则资金向消费、医药等防御性行业集中 |
| CPI | 居民消费价格指数 | 温和通胀(2-3%)利好股市；通缩(<0%)反映需求不足，利空；高通胀(>5%)引发紧缩担忧 |
| PPI | 工业生产者出厂价格指数 | PPI上行利好上游资源品；PPI与CPI剪刀差扩大意味着中下游制造业利润承压 |
| PMI | 采购经理人指数 | PMI>50为扩张区间，利好；PMI<50为收缩区间，利空。连续3个月下行趋势比绝对值更重要 |
| 社融 | 社会融资规模 | 社融超预期通常利好；社融结构（政府债占比vs企业中长期贷款）比总量更重要 |

**A股2024年实例**：2024年9月24日，央行、金融监管总局、证监会联合发布一系列政策（降准、降息、创设互换便利和股票回购增持再贷款），市场将此解读为宏观政策的重大转向。上证指数在随后5个交易日内从约2750点飙升至约3350点，涨幅超20%。这再次印证了政策面对A股短期走势的支配性影响——A股素有"政策市"之称。

#### 7.4 货币政策与财政政策

**货币政策（央行主导）：**

| 工具 | 操作方向 | 对股市的影响路径 |
|------|----------|-------------------|
| 降准 | 降低法定存款准备金率 | 释放银行可贷资金 → 市场流动性改善 → 利好 |
| 降息 | 降低政策利率（MLF/LPR） | 降低企业融资成本 → 利润改善 + 估值提升 → 利好 |
| 公开市场操作 | 逆回购/MLF投放 | 短期流动性调节，影响市场情绪 |

货币政策通过两条渠道影响股市：
- **流动性渠道**：货币宽松 → 市场资金充裕 → 部分资金流入股市。
- **估值渠道**：利率下降 → 折现率降低 → DCF估值中的企业价值上升（尤其是对利率敏感的成长股）。

**财政政策（财政部主导）：**

| 工具 | 操作方向 | 受益行业 |
|------|----------|----------|
| 减税降费 | 降低企业所得税/增值税 | 全市场，尤其中小企业 |
| 增加支出 | 基建投资、政府采购 | 建筑建材、工程机械、新能源 |
| 专项债 | 地方政府专项债券发行 | 基建、城市更新相关产业链 |

货币政策影响市场整体水位（贝塔），财政政策影响结构性机会（阿尔法）。投资者应关注"政策组合拳"——货币与财政协同发力时，对市场的推动力最为显著。

> **一句话总结：A股是政策敏感性市场——货币政策决定水位高低，财政政策决定水流方向。**

## 结语：技术分析与基本面分析的统一框架

本手册分别介绍了技术分析与基本面分析的方法论。在实际投资中，两者的关系并非对立而是互补：

- **基本面分析**回答"买什么"——筛选出具有竞争优势（宽护城河）、估值合理、处于成长或成熟期行业的优质企业。
- **技术分析**回答"什么时候买"——在基本面确认标的价值的基础上，利用技术信号寻找风险收益比更优的入场和离场时机。
- **风险管理**是两者的共同前提——仓位控制、止损纪律、组合分散化，是长期生存的基础。

最后再次强调本手册开篇的原则：**技术分析不是万能药，基本面分析也不能提供精确的买卖点。** 市场短期是投票机（情绪驱动），长期是称重机（价值驱动）。理解这两套语言，是为了在投票机疯狂时保持冷静，在称重机准确时保持耐心。
