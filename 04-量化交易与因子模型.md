## 第1章 量化交易全景图

### 1.1 什么是量化交易

想象你每天开车上班。有的人喜欢自己握着方向盘，凭感觉判断何时加速、何时变道、何时刹车——这是**主观交易**。而量化交易，相当于给汽车装上一套**自动驾驶系统**：传感器（数据）持续采集路况信息，算法（模型）实时计算最优策略，执行器（交易系统）毫秒级完成操作，全程无需人工干预。

用一句话定义：

$$ \text{量化交易} = \text{数据} + \text{模型} + \text{纪律} $$

- **数据（Data）：** 量化的"眼睛"。价格、成交量、财务数据、另类数据（卫星图像、社交媒体情绪），一切可被数字化的信息都是原材料。
- **模型（Model）：** 量化的"大脑"。从简单的线性回归到复杂的深度神经网络，模型从数据中提炼规律，形成交易信号。
- **纪律（Discipline）：** 量化的"灵魂"。机器不会恐惧、不会贪婪、不会因为昨晚没睡好而乱下单。规则一旦设定，严格执行。

传统主观交易者研究一家公司可能覆盖数十只股票，而量化系统可以同时监控全市场 5000+ 只股票，在人类眨眼的瞬间完成信号更新。这不是魔法的胜利，而是**算力对注意力的降维打击**。

### 1.2 核心流程

量化交易的完整生命周期是一条精密的生产流水线：

```
数据采集 → 数据清洗 → 因子计算 → 模型训练 → 回测验证 → 模拟交易 → 实盘执行
```

| 环节 | 核心任务 | 常见工具 |
|------|----------|----------|
| **数据采集** | 从行情源、财报库、另类数据商获取原始数据 | Akshare, Tushare, Wind, Quandl |
| **数据清洗** | 处理缺失值、异常值、复权调整、停牌标记 | Pandas, NumPy |
| **因子计算** | 将原始数据转化为标准化因子值（如PE倒数、动量） | Pandas, NumPy, Alphalens |
| **模型训练** | 用历史数据拟合因子与未来收益的关系 | Scikit-learn, LightGBM, PyTorch |
| **回测验证** | 在历史行情中模拟交易，评估策略表现 | Backtrader, Zipline, 自研框架 |
| **模拟交易** | 在实时行情但虚拟资金的环境中"彩排" | 券商模拟盘, 自研 Paper Trading |
| **实盘执行** | 信号通过 API 送达券商，真实资金成交 | CTP, XTP, 券商 API |

每一步都不可跳过。数据清洗不充分，后面一切都是"垃圾进、垃圾出"；回测草率了事，实盘亏损就是学费。

### 1.3 量化策略分类

#### （一）统计套利

核心思想：**两个高度相关的资产，价差偏离均值后会回归。**

最经典的例子是**配对交易**。以 A 股为例，贵州茅台（600519）和五粮液（000858）同属高端白酒板块，历史上价格走势高度相关。当两者价差扩大到 2 个标准差以外时，做多被低估的一方、做空被高估的一方，等价差回归后平仓获利。

$$\text{信号} = \frac{P_A - \beta \cdot P_B - \mu}{\sigma}$$

当 $|\text{信号}| > 2$ 时触发交易。

#### （二）趋势跟踪

核心思想：**趋势一旦形成，会持续一段时间。** "截断亏损，让利润奔跑。"

最经典的实现是**海龟交易法则**（Richard Dennis, 1983），用唐奇安通道突破入场，ATR 动态管理仓位，金字塔加仓。下面是一个简化实现：

```python
"""
海龟交易法则 - 简化版趋势跟踪策略
教育目的演示，不构成投资建议
"""
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 模拟价格数据 (或替换为 akshare 获取真实数据)
np.random.seed(42)
n_days = 500
returns = np.random.randn(n_days) * 0.015
returns[50:80] += 0.008    # 模拟一波上涨趋势
returns[200:250] -= 0.008  # 模拟一波下跌趋势
price = 100 * np.exp(np.cumsum(returns))
df = pd.DataFrame({'close': price}, index=pd.date_range('2024-01-01', periods=n_days))

# ========== 海龟策略参数 ==========
donchian_entry = 20  # 突破周期
donchian_exit = 10   # 退出周期
atr_period = 20      # ATR 计算周期
N_multiplier = 2     # 止损倍数

# 计算指标
df['high_entry'] = df['close'].rolling(donchian_entry).max()
df['low_entry']  = df['close'].rolling(donchian_entry).min()
df['high_exit']  = df['close'].rolling(donchian_exit).max()
df['low_exit']   = df['close'].rolling(donchian_exit).min()

# ATR (Average True Range 简化版)
df['atr'] = df['close'].diff().abs().rolling(atr_period).mean()

# ========== 生成信号 ==========
df['signal'] = 0
# 入场：突破20日高点做多
df.loc[df['close'] > df['high_entry'].shift(1), 'signal'] = 1
# 出场：跌破10日低点平仓
df.loc[df['close'] < df['low_exit'].shift(1), 'signal'] = -1

# ========== 模拟收益 ==========
df['position'] = df['signal'].replace(0, np.nan).ffill().fillna(0)
df['strategy_return'] = df['position'].shift(1) * df['close'].pct_change()

# ========== 可视化 ==========
fig, axes = plt.subplots(3, 1, figsize=(14, 10), sharex=True)

axes[0].plot(df.index, df['close'], color='#2c3e50', linewidth=1.2, label='价格')
axes[0].plot(df.index, df['high_entry'], 'g--', alpha=0.6, label=f'{donchian_entry}日高点')
axes[0].plot(df.index, df['low_entry'], 'r--', alpha=0.6, label=f'{donchian_entry}日低点')
axes[0].set_ylabel('价格', fontsize=11)
axes[0].legend(loc='upper left', fontsize=9)
axes[0].set_title('海龟交易法则 - 唐奇安通道策略 (教育演示)', fontsize=14, fontweight='bold')

axes[1].fill_between(df.index, 0, df['signal'], color='steelblue', alpha=0.4, step='post')
axes[1].set_ylabel('信号', fontsize=11)
axes[1].set_ylim(-1.5, 1.5)
axes[1].set_yticks([-1, 0, 1])
axes[1].set_yticklabels(['空仓', '空仓', '做多'])

cumulative = (1 + df['strategy_return']).cumprod()
axes[2].plot(df.index, cumulative, color='#e74c3c', linewidth=1.5, label='策略净值')
axes[2].axhline(1, color='gray', linestyle='--', alpha=0.5)
axes[2].set_ylabel('策略净值', fontsize=11)
axes[2].set_xlabel('日期', fontsize=11)
axes[2].legend(loc='upper left', fontsize=9)

plt.tight_layout()
plt.savefig('turtle_strategy.png', dpi=150, bbox_inches='tight')
plt.show()
print("注意：以上为历史模拟结果，实盘中需考虑滑点、手续费、流动性等成本。")
```

#### （三）高频交易（HFT）

在微秒（百万分之一秒）级别完成报价、成交、撤单的循环。典型策略包括做市商（赚买卖价差）、跨交易所套利、订单流预测。门槛极高：需要交易所机房托管、FPGA 硬件加速、低延迟网络。**对于个人投资者，这条路基本走不通**，资金和技术壁垒远超想象。

#### （四）因子投资（重点）

这是本章的核心。与其预测每只股票的涨跌，不如找到**驱动收益的系统性因子**，构建因子组合获取长期 Alpha。它的逻辑链条是：

$$ \text{找到可量化的股票特征} \rightarrow \text{验证与收益的统计关系} \rightarrow \text{构建分散化组合} $$

Fama-French 三因子模型（1993）奠定了这一领域的理论基础，如今业界使用的因子数量可能多达数百个。

### 1.4 优势与陷阱

| 维度 | 优势 | 陷阱 |
|------|------|------|
| **客观性** | 不受情绪干扰，信号即行动 | 过拟合：在历史数据上表现完美，实盘一塌糊涂 |
| **可复制性** | 策略代码化，可反复执行 | 幸存者偏差：只看到活下来的策略，忽略了大量失败者 |
| **可回测性** | 历史数据可量化验证 | 未来信息：不经意间使用了当时不可知的数据 |
| **广度** | 同时覆盖数千只标的 | 模型衰减：市场会进化，旧因子会失效 |
| **风控** | 规则化止损止盈 | 黑天鹅：模型从未见过的极端行情 |

> **最重要的提醒：** 回测中夏普比率 3.0 的策略，实盘中可能只有 0.5。差距来自：过拟合、手续费未计入、冲击成本忽略、未来信息泄露。做量化，诚实是第一原则——**永远怀疑自己的回测结果**。

## 第2章 Alpha 与 Beta

### 2.1 Beta —— 市场给你的收益

假设沪深 300 今天涨了 1%，你的持仓涨了 1.2%。多出来的 0.2% 是你的选股能力，而那 1% 只是"水涨船高"——市场涨你也涨，这叫 **Beta 收益**。

正式定义：Beta 衡量单只股票对市场组合（通常用沪深 300 或中证 800 代替）的敏感度。

$$ R_i - R_f = \alpha_i + \beta_i (R_m - R_f) + \varepsilon_i $$

其中：
- $R_i$：股票 i 的收益率
- $R_f$：无风险利率（可用国债收益率或 shibor）
- $R_m$：市场组合收益率
- $\beta_i$：股票 i 的系统性风险暴露
- $\alpha_i$：超额收益（截距项）
- $\varepsilon_i$：残差（个股特有波动）

**用 Python 计算单只股票的 Beta：**

```python
"""
计算单只股票的 Beta 系数
教育目的演示，数据为模拟数据，实际可替换为 akshare/Tushare 真实数据
"""
import pandas as pd
import numpy as np
import statsmodels.api as sm
import matplotlib.pyplot as plt

# ========== 模拟数据 ==========
# 实际使用时替换为真实数据，例如：
# import akshare as ak
# stock = ak.stock_zh_a_hist(symbol="600519", period="daily", start_date="20230101")
# index = ak.stock_zh_index_daily(symbol="sh000300")

np.random.seed(42)
n = 252  # 一年的交易日
market_return = np.random.normal(0.0003, 0.012, n)  # 市场日均收益约0.03%，波动1.2%

# 假设股票 Beta=1.2，alpha=0.0001 (日度)
true_beta = 1.2
true_alpha = 0.0001
stock_return = true_alpha + true_beta * market_return + np.random.normal(0, 0.008, n)

df = pd.DataFrame({
    'market': market_return,
    'stock': stock_return
}, index=pd.date_range('2024-01-01', periods=n, freq='B'))

# ========== OLS 回归计算 Beta ==========
X = sm.add_constant(df['market'])  # 添加截距项
y = df['stock']
model = sm.OLS(y, X).fit()

alpha_cap = model.params['const']
beta_cap = model.params['market']
r_squared = model.rsquared

print(f"{'='*50}")
print(f"  单只股票 Beta 计算结果")
print(f"{'='*50}")
print(f"  估计 Alpha (日度):  {alpha_cap:.6f}  (真实: {true_alpha:.6f})")
print(f"  估计 Beta:          {beta_cap:.3f}       (真实: {true_beta:.1f})")
print(f"  R-squared:          {r_squared:.3f}")
print(f"  解释: 该股票约 {beta_cap:.2f} 倍跟随市场波动，")
print(f"        市场涨1%时，预期该股涨约 {beta_cap:.2f}%")
print(f"{'='*50}")

# ========== 可视化 ==========
fig, ax = plt.subplots(figsize=(9, 7))
ax.scatter(df['market'] * 100, df['stock'] * 100, alpha=0.5,
           color='#3498db', edgecolors='white', s=40, label='每日收益散点')
x_range = np.linspace(df['market'].min(), df['market'].max(), 100) * 100
ax.plot(x_range, alpha_cap * 100 + beta_cap * x_range,
        color='#e74c3c', linewidth=2, label=f'拟合线 (Beta={beta_cap:.2f})')
ax.axhline(0, color='gray', linestyle='--', alpha=0.5)
ax.axvline(0, color='gray', linestyle='--', alpha=0.5)
ax.set_xlabel('市场收益率 (%)', fontsize=12)
ax.set_ylabel('股票收益率 (%)', fontsize=12)
ax.set_title(f'Beta 回归分析 — 市场 vs 个股收益率', fontsize=14, fontweight='bold')
ax.legend(fontsize=10)

# 添加文本框显示结果
textstr = f'Alpha (日) = {alpha_cap*100:.4f}%\nBeta = {beta_cap:.3f}\n$R^2$ = {r_squared:.3f}'
props = dict(boxstyle='round', facecolor='wheat', alpha=0.8)
ax.text(0.05, 0.95, textstr, transform=ax.transAxes, fontsize=11,
        verticalalignment='top', bbox=props)

plt.tight_layout()
plt.savefig('beta_regression.png', dpi=150, bbox_inches='tight')
plt.show()
```

**Beta 的实际意义：**

- $\beta = 1$：与市场同涨同跌（如沪深 300 ETF）
- $\beta > 1$：市场放大器，弹性大（如券商股、科技股）
- $\beta < 1$：防御型，波动小（如公用事业、消费必需品）
- $\beta < 0$：理论上与市场反向，实际极少见

### 2.2 Alpha —— 你超越市场的部分

Alpha 是剥离了市场 Beta 收益后，剩余的超额收益。来源有三类：

1. **定价错误（Mispricing）：** 市场并非完全有效。投资者情绪、流动性冲击、指数调仓等都会制造短暂的错误定价，量化模型可以捕捉。
2. **行为偏差（Behavioral Bias）：** 投资者过度反应（追涨杀跌）和反应不足（对基本面变化视而不见），创造了可预测的收益模式。
3. **信息优势（Information Advantage）：** 更快的资讯处理、更全面的另类数据覆盖、更准确的另类数据解读——这是量化基金的护城河。

在 A 股市场，由于个人投资者占比仍较高（约 60%），行为偏差更加显著，为 Alpha 策略提供了更肥沃的土壤。但监管环境、政策风险也是必须认真对待的因素。

### 2.3 信息比率（Information Ratio）

有了 Alpha，下一个问题是：**这个 Alpha 稳定吗？** 信息比率 IR 回答了这个问题。

$$ \text{IR} = \frac{\text{年化超额收益}}{\text{年化跟踪误差}} = \frac{\bar{\alpha} \times 252}{\sigma_\alpha \times \sqrt{252}} $$

其中 $\bar{\alpha}$ 是日均超额收益，$\sigma_\alpha$ 是超额收益的标准差。IR 衡量的是**单位主动风险所获得的超额回报**。

- IR < 0：策略跑输基准，或者需要重新审视
- IR 0 ~ 0.3：勉强及格，可能仅是运气
- IR 0.3 ~ 0.5：还可以，有一定选股能力
- IR 0.5 ~ 1.0：不错，专业投资者水平
- IR > 1.0：很好，顶级量化基金水准

```python
"""
信息比率 (Information Ratio) 计算
教育目的演示
"""
import pandas as pd
import numpy as np

# ========== 模拟策略超额收益序列 ==========
np.random.seed(123)
n_days = 252 * 3  # 三年

# 模拟三种不同能力水平的策略
# 策略A: IR≈0.3 (入门)
# 策略B: IR≈0.7 (不错)
# 策略C: IR≈1.2 (优秀)
alpha_daily_a = np.random.normal(0.0004, 0.006, n_days)  # 日均4bp, 波动60bp
alpha_daily_b = np.random.normal(0.0006, 0.004, n_days)  # 日均6bp, 波动40bp
alpha_daily_c = np.random.normal(0.0008, 0.003, n_days)  # 日均8bp, 波动30bp

# ========== 计算 IR 的函数 ==========
def calculate_ir(alpha_daily, trading_days=252):
    """
    计算年化信息比率

    参数:
        alpha_daily: 日度超额收益序列
        trading_days: 年交易日数
    """
    avg_alpha = np.mean(alpha_daily)
    std_alpha = np.std(alpha_daily, ddof=1)
    annual_alpha = avg_alpha * trading_days
    annual_te = std_alpha * np.sqrt(trading_days)
    ir = annual_alpha / annual_te if annual_te > 0 else 0
    return {
        '日均超额收益': avg_alpha,
        '超额收益波动率': std_alpha,
        '年化超额收益': annual_alpha,
        '年化跟踪误差': annual_te,
        '信息比率 (IR)': ir
    }

# ========== 输出结果 ==========
for name, alpha in [('策略A (入门)', alpha_daily_a),
                     ('策略B (不错)', alpha_daily_b),
                     ('策略C (优秀)', alpha_daily_c)]:
    result = calculate_ir(alpha)
    print(f"\n{'─'*45}")
    print(f"  {name}")
    print(f"{'─'*45}")
    print(f"  日均超额收益:   {result['日均超额收益']:.5f} ({result['日均超额收益']*100:.2f}%)")
    print(f"  超额波动率:     {result['超额收益波动率']:.5f} ({result['超额收益波动率']*100:.2f}%)")
    print(f"  年化超额收益:   {result['年化超额收益']:.2%}")
    print(f"  年化跟踪误差:   {result['年化跟踪误差']:.2%}")
    print(f"  信息比率 (IR):  {result['信息比率 (IR)']:.2f}")
    ir_val = result['信息比率 (IR)']
    if ir_val < 0.3:
        level = '需要改进'
    elif ir_val < 0.7:
        level = '入门水平'
    elif ir_val < 1.0:
        level = '不错，专业水准'
    else:
        level = '优秀，顶尖水平'
    print(f"  评级:           {level}")

# ========== 可视化 ==========
import matplotlib.pyplot as plt

fig, axes = plt.subplots(3, 1, figsize=(12, 9))
colors = ['#3498db', '#2ecc71', '#e74c3c']

for i, (name, alpha, color) in enumerate(zip(
    ['策略A (IR≈0.3)', '策略B (IR≈0.7)', '策略C (IR≈1.2)'],
    [alpha_daily_a, alpha_daily_b, alpha_daily_c],
    colors)):
    cumulative = (1 + alpha).cumprod()
    axes[i].plot(cumulative, color=color, linewidth=1, alpha=0.8)
    ir = calculate_ir(alpha)['信息比率 (IR)']
    axes[i].set_title(f'{name} | IR = {ir:.2f}', fontsize=12, fontweight='bold')
    axes[i].set_ylabel('累计超额净值')
    axes[i].axhline(1, color='gray', linestyle='--', alpha=0.4)

axes[-1].set_xlabel('交易日')
plt.suptitle('不同 IR 水平的策略超额收益路径', fontsize=14, fontweight='bold')
plt.tight_layout()
plt.savefig('information_ratio.png', dpi=150, bbox_inches='tight')
plt.show()

print("\n注意：高 IR 不等于高收益。一个年化超额2%但波动仅1%的策略")
print("IR=2.0，可能比年化超额20%但波动30%的策略(IR=0.67)更"可持续"。")
```

## 第3章 因子模型

### 3.1 什么是因子

> **因子（Factor）** = 一只股票可以被量化的某种特征，且该特征对未来的截面收益有统计上显著的预测能力。

举个例子：把全市场股票按市盈率（PE）从低到高排好队。发现一个规律——PE 最低的那组股票，未来一年的平均收益比 PE 最高的那组高出 15 个百分点。这个现象反复出现，不是偶然。那么我们说，**低 PE 是一个"价值因子"**。

因子的本质是对收益来源的**归因拆解**。就像物理学家把白光分解成七彩光谱，量化研究员把收益分解为估值因子、动量因子、质量因子……每只股票都可以用一串因子分数来表示：

$$ \text{股票}_i = \text{Beta敞口}_i + \text{估值敞口}_i + \text{动量敞口}_i + \text{规模敞口}_i + \text{质量敞口}_i + \cdots $$

### 3.2 经典因子详解

下面逐一拆解量化领域最经典的八大类因子。每个因子都会给出核心逻辑、Python 计算代码以及 A 股有效性讨论。

---

#### A. 动量因子（Momentum）

**核心逻辑：** 过去 6~12 个月涨幅靠前的股票，未来 1~3 个月继续跑赢的概率更大。注意，需要剔除最近 1 个月的收益，因为短期存在反转效应（见 B）。这是行为金融中"反应不足"的体现——好消息出来后，市场没有一次性定价到位。

$$ \text{Momentum} = \prod_{t=T-12}^{T-1} (1 + r_t) - 1 \quad \text{（剔除最近1个月）} $$

```python
"""
动量因子计算
剔除最近1个月，使用过去2-12个月累计收益
"""
import pandas as pd
import numpy as np

# ========== 模拟股票价格数据 ==========
np.random.seed(42)
n_stocks = 10
n_days = 252 * 2  # 两年数据

# 生成相关但有差异的股票价格
dates = pd.date_range('2023-01-01', periods=n_days, freq='B')
prices_dict = {}
for i in range(n_stocks):
    drift = np.random.uniform(-0.05, 0.15) / 252
    vol = np.random.uniform(0.015, 0.035)
    returns = drift + vol * np.random.randn(n_days)
    prices_dict[f'股票_{i+1}'] = 100 * np.exp(np.cumsum(returns))

df_price = pd.DataFrame(prices_dict, index=dates)

# ========== 动量因子计算 ==========
def calc_momentum(price_df, skip_month=1, lookback_months=11):
    """
    计算动量因子：过去N个月收益（剔除最近skip_month个月）

    参数:
        price_df: 价格 DataFrame, 行=日期, 列=股票
        skip_month: 剔除最近几个月 (通常是1)
        lookback_months: 回溯月数 (通常是11, 加上剔除的1个月 = 12个月窗口)
    """
    monthly_price = price_df.resample('M').last()

    # 计算月度收益
    monthly_ret = monthly_price.pct_change()

    # 动量 = 过去lookback_months个月累积收益 (剔除最近skip_month)
    momentum = monthly_ret.shift(skip_month).rolling(lookback_months).apply(
        lambda x: np.prod(1 + x) - 1, raw=False
    )

    # 取最新一期的因子值
    latest_momentum = momentum.iloc[-1]
    return latest_momentum.dropna()

momentum_scores = calc_momentum(df_price)
print("动量因子得分（最新一期）：")
print(momentum_scores.sort_values(ascending=False).to_string())
print("\n解释：数值越大，过去12个月（剔除最近1月）涨幅越大，动量越强。")
```

**A 股有效性：** 动量效应在 A 股中长期存在但并非单调。2015 年前动量效应较弱甚至有反向特征，2016 年后随着机构化加速，动量因子有效性明显改善。**注意**：A 股的动量效应在中小盘中更强，大盘股动量偏弱。

#### B. 反转因子（Reversal）

**核心逻辑：** 短期（1 天 ~ 1 周）涨幅过大的股票，很快会回落。这是市场微观结构中流动性冲击和过度反应的体现。买方力量短期集中释放后，股价暂时"透支"理性价位。

$$ \text{Reversal} = -\frac{P_{T} - P_{T-k}}{P_{T-k}} \quad (k \in [1, 5] 日) $$

```python
"""
短期反转因子计算
过去5日累计收益取负值（越高说明跌得越多，预期反弹）
"""
# 使用上面生成的 df_price

def calc_reversal(price_df, window=5):
    """
    计算短期反转因子：过去window日收益取负数
    负值越大 → 过去涨得多 → 预期反转下跌
    正数越大 → 过去跌得多 → 预期反转上涨
    """
    ret = price_df.pct_change(window)
    reversal = -ret  # 取负值，使"跌得多"得高分
    latest_reversal = reversal.iloc[-1]
    return latest_reversal.dropna()

reversal_scores = calc_reversal(df_price, window=5)
print("反转因子得分（最新一期）：")
print(reversal_scores.sort_values(ascending=False).to_string())
print("\n解释：正值越大 → 过去跌得越多 → 反转上涨预期越强。")
```

**A 股有效性：** 短期反转在 A 股非常显著，尤其在小盘股和散户主导的标的中。**散户追涨杀跌的行为为反向交易者提供了利润。** 但需警惕"接飞刀"——有些下跌不是短期情绪，而是基本面恶化。

#### C. 市值因子（Size）

**核心逻辑：** 全球范围内，小市值股票长期收益高于大市值股票。这被称为"规模溢价"。然而近十年在大盘科技股的冲击下，这一效应有所减弱。在 A 股，小市值效应历史上极为显著（2017年前），但 2017-2020 年大盘蓝筹牛市中一度失效，2021 年后又有所回归。

$$ \text{Size} = -\log(\text{TotalMarketCap}) $$

```python
"""
市值因子计算
小市值得分高，大市值得分低
"""
# 模拟市值数据
np.random.seed(456)
n_stocks = 10
# 对数正态分布模拟市值（单位：亿元）
market_cap = np.random.lognormal(mean=np.log(500), sigma=1.5, size=n_stocks)
stock_names = [f'股票_{i+1}' for i in range(n_stocks)]

df_mcap = pd.DataFrame({
    '股票名称': stock_names,
    '总市值(亿)': market_cap.round(1),
    '流通市值(亿)': (market_cap * np.random.uniform(0.6, 0.95, n_stocks)).round(1)
})

# 市值因子 = -log(市值)，市值越小 → 因子值越大
df_mcap['市值因子'] = -np.log(df_mcap['总市值(亿)'])

# 按因子值排序（从高到低 = 小盘到大盘）
df_mcap_sorted = df_mcap.sort_values('市值因子', ascending=False)
print("市值因子得分（越小盘 → 得分越高）：")
print(df_mcap_sorted.to_string(index=False))
```

**A 股有效性：** 历史上最有效的 A 股因子之一，但**波动极大**，小盘组合在熊市中跌幅也更深。建议将市值作为风险因子控制，而非单纯做多。

#### D. 价值因子（Value）

**核心逻辑：** 便宜货最终会回归合理估值。使用市盈率（PE）、市净率（PB）、市销率（PS）的倒数来衡量"便宜程度"。**但必须警惕价值陷阱**——有些股票 PE 低是因为基本面在恶化，股价已经提前反映。

$$ \text{Value}_i = \frac{1}{3} \left(\frac{1}{PE_i} + \frac{1}{PB_i} + \frac{1}{PS_i}\right) $$

```python
"""
价值因子计算
使用 EP(=1/PE)、BP(=1/PB)、SP(=1/PS) 构建综合价值因子
"""
# 模拟财务数据
np.random.seed(789)
n = 20

df_value = pd.DataFrame({
    '股票': [f'股票_{i+1}' for i in range(n)],
    'PE': np.random.uniform(8, 80, n),    # 市盈率
    'PB': np.random.uniform(0.8, 8, n),   # 市净率
    'PS': np.random.uniform(0.5, 15, n),  # 市销率
})

# 计算因子：EP, BP, SP (取倒数，使高估值→低得分, 低估值→高得分)
df_value['EP因子'] = 1 / df_value['PE']
df_value['BP因子'] = 1 / df_value['PB']
df_value['SP因子'] = 1 / df_value['PS']

# 先对每个子因子做截面标准化（Z-Score），再加权合成
for col in ['EP因子', 'BP因子', 'SP因子']:
    df_value[f'{col}_z'] = (df_value[col] - df_value[col].mean()) / df_value[col].std()

# 等权合成综合价值因子
df_value['综合价值因子'] = (df_value['EP因子_z'] + df_value['BP因子_z'] + df_value['SP因子_z']) / 3

print("价值因子得分（高得分 = 低估值 = 便宜）：")
display_cols = ['股票', 'PE', 'PB', 'PS', '综合价值因子']
print(df_value[display_cols].sort_values('综合价值因子', ascending=False).round(2).to_string(index=False))

print("\n" + "="*60)
print("  ⚠ 警惕价值陷阱：PE极低的股票可能处于基本面恶化中。")
print("  建议结合质量因子（ROE/毛利率）过滤价值陷阱。")
print("="*60)
```

**A 股有效性：** 价值因子在 A 股总体有效，但呈现明显**阶段性**：牛市末期和熊市中价值因子表现更好，牛市中前期则成长风格占优。银行股（天然低 PE）需要行业中性化处理。

#### E. 质量因子（Quality）

**核心逻辑：** 好公司长期会创造好回报。用高 ROE（净资产收益率）、高毛利率、低负债率来定义"好公司"。A 股尤其看重 ROE——这与中国投资者偏好盈利确定性的文化有关。

$$ \text{Quality} = w_1 \cdot \text{ROE} + w_2 \cdot \text{GrossMargin} - w_3 \cdot \text{DebtRatio} $$

```python
"""
质量因子计算
ROE高 + 毛利率高 + 负债率低 = 高质量
"""
np.random.seed(101)
n = 15

df_quality = pd.DataFrame({
    '股票': [f'股票_{i+1}' for i in range(n)],
    'ROE(%)': np.random.uniform(2, 35, n),        # 净资产收益率
    '毛利率(%)': np.random.uniform(10, 80, n),     # 毛利率
    '资产负债率(%)': np.random.uniform(15, 75, n), # 负债率
})

# Z-Score 标准化
for col in ['ROE(%)', '毛利率(%)', '资产负债率(%)']:
    df_quality[f'{col}_z'] = (df_quality[col] - df_quality[col].mean()) / df_quality[col].std()

# 综合质量因子：ROE+毛利率为正贡献，负债率为负贡献
df_quality['质量因子'] = (
    df_quality['ROE(%)_z'] * 0.4 +
    df_quality['毛利率(%)_z'] * 0.3 -
    df_quality['资产负债率(%)_z'] * 0.3
)

print("质量因子得分（高分 = 盈利好 + 毛利高 + 负债低）：")
display = ['股票', 'ROE(%)', '毛利率(%)', '资产负债率(%)', '质量因子']
print(df_quality[display].sort_values('质量因子', ascending=False).round(2).to_string(index=False))
```

**A 股有效性：** 质量因子近年来表现优异，尤其是外资（北向资金）偏好的标的。ROE 是 A 股最稳健的单因子之一。2019-2021 年核心资产行情中，质量因子大幅跑赢。

#### F. 波动率因子（Volatility / Low Beta）

**核心逻辑：** 这是一个反直觉的发现——**低波动率的股票，长期收益反而高于高波动率股票。** 原因包括：杠杆约束（想追求高收益但无法加杠杆的投资者倾向于买高波动股票，推高价格拉低未来收益）、彩票偏好（投资者喜欢"以小博大"）。

$$ \text{Volatility} = -\sigma(R_{i, \text{过去60日}}) $$

```python
"""
低波动因子计算
低波动 → 高分，高波动 → 低分
"""
# 使用前面生成的 df_price
ret_df = df_price.pct_change().dropna()

# 计算过去60个交易日的波动率
volatility = ret_df.rolling(60).std().iloc[-1]

# 低波动因子 = -波动率
low_vol_factor = -volatility

print("低波动因子得分（低波动 → 高分）：")
print(low_vol_factor.sort_values(ascending=False).to_string())
print("\n解释：因子值越大，说明过去60日波动越低，")
print("      从"低波动异象"来看，预期未来收益更高。")
print("      ⚠ 这个效应在A股的稳定性不如美股，需谨慎使用。")
```

**A 股有效性：** 低波动异象在 A 股存在但不稳定。A 股投资者散户比例高，"涨停板敢死队"文化根深蒂固，高波动的投机吸引力削弱了低波动溢价。建议作为辅助因子使用。

#### G. 情绪因子（Sentiment）

**核心逻辑：** 市场参与者的情绪会影响短期价格。传统情绪指标包括分析师评级上调比例、新闻舆情得分、社交媒体讨论热度。**近年来，大语言模型（LLM）在处理非结构化文本（财报电话会、新闻、社交媒体）方面的能力，为情绪因子开发打开了全新空间。**

```python
"""
情绪因子 — 模拟分析师评级变化
实际应用中可接入 NLP 模型分析新闻/社交媒体文本
"""
np.random.seed(303)
n = 12

df_sentiment = pd.DataFrame({
    '股票': [f'股票_{i+1}' for i in range(n)],
    '上调分析师数': np.random.randint(0, 8, n),
    '下调分析师数': np.random.randint(0, 5, n),
    '新闻正面率(%)': np.random.uniform(20, 85, n),
})

# 情绪综合得分
df_sentiment['情绪因子'] = (
    (df_sentiment['上调分析师数'] - df_sentiment['下调分析师数']) / 5 +
    df_sentiment['新闻正面率(%)'] / 50
)

print("情绪因子得分（高分 = 正面情绪）：")
print(df_sentiment.sort_values('情绪因子', ascending=False).to_string(index=False))
print("\n💡 LLM应用场景：使用 GPT/DeepSeek 等模型分析财报文本，")
print("   提取管理层语气、风险披露变化、业绩指引倾向等细粒度信号。")
```

**A 股有效性：** A 股中分析师覆盖相对集中（大市值公司），情绪因子在这部分标的中效果较好。LLM 赋能的另类情绪分析是新蓝海，**但需要警惕：这些信号衰减很快，必须持续迭代模型。**

#### H. 技术因子（Technical）

**核心逻辑：** 基于价格、成交量的技术指标在某些市场阶段仍有预测力。MACD 金叉死叉、RSI 超买超卖、布林带突破是最经典的三个。

```python
"""
技术因子 — RSI 与 布林带位置
教育目的演示
"""
# 使用前面生成的 df_price 取一只股票
stock_price = df_price['股票_1'].values

def calc_rsi(prices, period=14):
    """计算 RSI 指标"""
    deltas = np.diff(prices)
    gains = np.where(deltas > 0, deltas, 0)
    losses = np.where(deltas < 0, -deltas, 0)
    avg_gain = pd.Series(gains).rolling(period).mean().iloc[-1]
    avg_loss = pd.Series(losses).rolling(period).mean().iloc[-1]
    if avg_loss == 0:
        return 100.0
    rs = avg_gain / avg_loss
    return 100 - 100 / (1 + rs)

def calc_bollinger_position(prices, period=20, num_std=2):
    """计算布林带位置 (0=下轨, 1=上轨)"""
    rolling_mean = pd.Series(prices).rolling(period).mean().iloc[-1]
    rolling_std = pd.Series(prices).rolling(period).std().iloc[-1]
    upper = rolling_mean + num_std * rolling_std
    lower = rolling_mean - num_std * rolling_std
    current = prices[-1]
    position = (current - lower) / (upper - lower)  # 0-1 之间的位置
    return position

rsi_val = calc_rsi(stock_price)
bb_pos = calc_bollinger_position(stock_price)

print(f"RSI(14):           {rsi_val:.1f}")
print(f"  RSI > 70 → 超买, RSI < 30 → 超卖")
print(f"布林带位置:        {bb_pos:.2f}")
print(f"  0 = 下轨附近(可能超卖反弹), 1 = 上轨附近(可能超买回调)")

# 批量计算所有股票的技术因子
print(f"\n全市场技术因子一览：")
for col in df_price.columns:
    prices = df_price[col].values
    rsi = calc_rsi(prices)
    bb = calc_bollinger_position(prices)
    print(f"  {col}: RSI={rsi:.1f}, 布林位置={bb:.2f}")
```

**A 股有效性：** 技术因子在 A 股的有效性**远高于成熟市场**，这与 A 股 60% 的个人投资者比例高度相关。技术分析在散户群体中普遍使用，形成自我实现的反馈循环。但单一技术指标单独使用效果有限，需组合使用。

## 第4章 因子检验方法论

找到一个候选因子后，要经过严格的统计检验才能纳入模型。标准流程包括 IC 分析、分层回测和相关性分析三大关卡。

### 4.1 IC 分析（Information Coefficient）

IC（信息系数）衡量因子值（今天）与未来收益（明天/下周/下月）的**截面相关性**。它是判断一个因子是否"有效"的最核心指标。

$$ \text{IC}_t = \text{corr}(\text{Factor}_{t-1}, \text{Return}_t) $$

通常使用 **Rank IC（斯皮尔曼等级相关系数）**，因为它对异常值更加稳健。

- |IC| < 0.02：因子效果微弱
- 0.02 < |IC| < 0.03：勉强可用
- 0.03 < |IC| < 0.05：不错，可作为基础因子
- 0.05 < |IC|：优秀因子
- 另外要看 **IC_IR = mean(IC) / std(IC)**，IR > 0.5 意味着因子稳定

```python
"""
因子 IC (Information Coefficient) 分析
完整计算 IC 序列 + 可视化
"""
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# ========== 模拟因子数据和未来收益 ==========
np.random.seed(42)
n_periods = 120  # 120 个截面期
n_stocks = 200

# 模拟三个因子（不同 IC 水平）
# 因子A: IC≈0.02 (弱)
# 因子B: IC≈0.05 (中等)
# 因子C: IC≈0.08 (强)
factor_a = np.random.randn(n_periods, n_stocks) * 1.0
factor_b = np.random.randn(n_periods, n_stocks) * 1.0
factor_c = np.random.randn(n_periods, n_stocks) * 1.0

# 生成与因子相关的未来收益
fwd_return = np.random.randn(n_periods, n_stocks) * 0.02  # 基础噪音
# 加入因子信号
fwd_return += 0.02 * 0.02 * factor_a   # IC≈0.02
fwd_return += 0.05 * 0.02 * factor_b   # IC≈0.05
fwd_return += 0.08 * 0.02 * factor_c   # IC≈0.08

# ========== IC 计算函数 ==========
def calc_ic_series(factor_df, return_df, method='rank'):
    """
    计算因子 IC 序列

    参数:
        factor_df: 因子值 DataFrame (行=时间, 列=股票)
        return_df: 未来收益 DataFrame (行=时间, 列=股票)
        method: 'pearson' 或 'rank' (斯皮尔曼)
    """
    ic_list = []
    for t in range(len(factor_df)):
        f = factor_df.iloc[t]
        r = return_df.iloc[t]
        # 移除 NaN
        valid = f.notna() & r.notna()
        if valid.sum() < 10:  # 至少需要10个有效数据点
            ic_list.append(np.nan)
            continue
        if method == 'rank':
            ic, _ = stats.spearmanr(f[valid], r[valid])
        else:

        ic_list.append(ic)
    return pd.Series(ic_list, index=factor_df.index)
```

### 4.2 分层回测（Quantile / Group Backtest）

分层回测是检验因子**单调性**的黄金标准。操作步骤：

1. 在每期截面上，按因子值将股票分成 N 组（通常 5 组或 10 组）
2. 每组等权持有，计算各组未来一段时间的收益
3. 理想情况下，收益应随因子值单调递增（或递减）
4. 重点关注 **Top - Bottom**（第一组减最后一组）的多空收益


## 第5章 多因子模型构建

有了多个经过检验的单因子，下一步是将它们合成为一个综合选股信号。

### 5.1 因子合成方法

三种主流加权方法：

1. **等权（Equal Weight）：** 最简单，所有因子权重相同。缺点是会给无效因子和有效因子一样的权重。

2. **IC 加权：** 因子权重 = 该因子历史 IC 均值。预测能力越强的因子权重越大。
   $$ w_i = \frac{\overline{\text{IC}}_i}{\sum |\overline{\text{IC}}_j|} $$

3. **ICIR 加权（推荐）：** 因子权重 = 该因子的 IC_IR。同时考虑预测能力和稳定性。
   $$ w_i = \frac{\text{IC_IR}_i}{\sum \text{IC_IR}_j} $$

```python
"""
因子合成 — 三种加权方法对比
"""
# 模拟各因子的历史 IC 统计
ic_stats = pd.DataFrame({
    '因子名称':   ['动量因子', '反转因子', '市值因子', '价值因子', '质量因子'],
    'IC均值':      [0.045,    0.038,    0.052,    0.041,    0.035],
    'IC标准差':    [0.12,     0.10,     0.14,     0.09,     0.08],
})
ic_stats['IC_IR'] = ic_stats['IC均值'] / ic_stats['IC标准差']

# 计算三种权重方案
n = len(ic_stats)
ic_stats['等权'] = 1 / n
ic_stats['IC加权'] = ic_stats['IC均值'] / ic_stats['IC均值'].sum()
ic_stats['ICIR加权'] = ic_stats['IC_IR'] / ic_stats['IC_IR'].sum()

print(f"{'='*70}")
print(f"  因子合成权重方案对比")
print(f"{'='*70}")
print(ic_stats[['因子名称', 'IC均值', 'IC_IR', '等权', 'IC加权', 'ICIR加权']].round(4).to_string(index=False))

print(f"\n  推荐使用 ICIR 加权，它同时考虑了因子的预测力和稳定性。")
print(f"  等权最简单但不区分因子质量；IC加权只看均值不考虑波动。")
```

### 5.2 完整选股流程

下面给出一个**端到端的完整多因子选股流程**，从因子计算到最终持仓输出：

```
计算所有因子 → 截面标准化 → 加权合成综合得分 → 选前20只 → 等权买入
```

```python
"""
多因子选股 — 完整流程
教育目的演示，使用模拟数据
"""
import pandas as pd
import numpy as np
from scipy import stats

np.random.seed(42)

# ========== 第1步：构建模拟数据 ==========
n_stocks = 100
stock_ids = [f'{i:06d}' for i in range(1, n_stocks + 1)]

# 行业分类（用于后续行业中性化）
industries = ['银行', '食品饮料', '医药', '电子', '计算机', '地产', '汽车', '化工']
industry_map = np.random.choice(industries, n_stocks)

# 模拟因子原始值
df_stocks = pd.DataFrame({
    '股票代码': stock_ids,
    '行业': industry_map,
    '动量因子_原始': np.random.randn(n_stocks) * 0.1 + 0.02,
    '反转因子_原始': np.random.randn(n_stocks) * 0.08,
    '市值因子_原始': np.random.lognormal(mean=np.log(500), sigma=1.5, size=n_stocks),  # 亿元
    '价值因子_原始': np.random.randn(n_stocks) * 0.15 + 0.01,
    '质量因子_原始': np.random.randn(n_stocks) * 0.12 + 0.03,
})

# ========== 第2步：因子处理 ==========
# 对市值因子取负对数（小盘=高得分）
df_stocks['市值因子_处理'] = -np.log(df_stocks['市值因子_原始'])

# 直接使用其他因子的原始值（实际中可能需要去极值、中性化等更多处理）
for factor in ['动量因子_原始', '反转因子_原始', '价值因子_原始', '质量因子_原始']:
    processed_name = factor.replace('_原始', '_处理')
    df_stocks[processed_name] = df_stocks[factor]

# ========== 第3步：截面标准化 (Z-Score) ==========
factor_cols = ['动量因子_处理', '反转因子_处理', '市值因子_处理',
               '价值因子_处理', '质量因子_处理']

for col in factor_cols:
    std_name = col.replace('_处理', '_zscore')
    df_stocks[std_name] = (df_stocks[col] - df_stocks[col].mean()) / df_stocks[col].std()

# ========== 第4步：加权合成综合得分 ==========
# 使用 ICIR 加权（示例用预设权重代替）

    '动量因子_zscore': 0.25,
    '反转因子_zscore': 0.15,
    '市值因子_zscore': 0.20,
    '价值因子_zscore': 0.25,
    '质量因子_zscore': 0.15,
}

df_stocks['综合得分'] = sum(df_stocks[col] * w for col, w in weights.items())

# ========== 第5步：选股 ==========
top_n = 20
df_stocks['排名'] = df_stocks['综合得分'].rank(ascending=False).astype(int)
df_portfolio = df_stocks[df_stocks['排名'] <= top_n].copy()
# 等权分配仓位
df_portfolio['权重'] = 1.0 / top_n

# ========== 输出结果 ==========
print(f"{'='*65}")
print(f"  多因子选股结果 — 综合得分前 {top_n} 名持仓")
print(f"{'='*65}")
display_cols = ['排名', '股票代码', '行业', '综合得分', '权重']

# 行业分布
print(f"\n  持仓行业分布：")
industry_dist = df_portfolio['行业'].value_counts()
for ind, cnt in industry_dist.items():
    print(f"    {ind}: {cnt} 只 ({cnt/top_n*100:.0f}%)")

# ========== 收益分布可视化 ==========
```

### 5.3 行业中性化

一个必须重视的问题：**不同行业的估值体系天然不同。** 银行股 PE 普遍 5-8 倍，科技股 PE 均值 30-50 倍。如果把银行和科技放一起比较 PE 因子，会得出"所有银行都很便宜、所有科技都很贵"的结论——这在行业中性化之前几乎毫无意义。

**行业中性化的标准做法：**

1. 在因子截面上，对行业哑变量做回归，取残差
2. 残差 = 因子中**除去行业影响的纯股票特异部分**
3. 或者在行业内部分别排名，用行业排名代替原始因子值

$$ \text{Factor}_i = \sum_j \beta_j \cdot \text{IndustryDummy}_{ij} + \varepsilon_i $$

$$ \text{Factor}_i^{\text{neutralized}} = \varepsilon_i $$

```python
"""
行业中性化处理
在行业内比较，去除行业系统性偏差
"""
from sklearn.linear_model import LinearRegression

# 使用前面 df_stocks 的数据
print(f"{'='*65}")
print(f"  行业中性化前后对比")
print(f"{'='*65}")

# ========== 方法1：回归取残差 ==========
# 对价值因子做行业中性化
industry_dummies = pd.get_dummies(df_stocks['行业'], drop_first=True)
X = industry_dummies.values
y = df_stocks['价值因子_zscore'].values

model = LinearRegression()
model.fit(X, y)
y_pred = model.predict(X)
df_stocks['价值因子_行业中性'] = y - y_pred  # 残差 = 中性化后的因子值

# ========== 方法2：行业内排名（Rank 方法，更稳健）==========
df_stocks['价值因子_行业排名'] = df_stocks.groupby('行业')['价值因子_zscore'].rank(pct=True)

# ========== 对比结果 ==========
print("\n银行股对比（行业内排名法解决了银行天然PE低的问题）：")
bank_stocks = df_stocks[df_stocks['行业'] == '银行'].head(5)
display_cols = ['股票代码', '行业', '价值因子_zscore', '价值因子_行业中性', '价值因子_行业排名']
print(bank_stocks[display_cols].round(4).to_string(index=False))

print("\n科技股对比：")
tech_stocks = df_stocks[df_stocks['行业'].isin(['电子', '计算机'])].head(5)
print(tech_stocks[display_cols].round(4).to_string(index=False))

# ========== 可视化 ==========
fig, axes = plt.subplots(1, 2, figsize=(13, 5.5))

ax = axes[0]
for ind in df_stocks['行业'].unique():
    subset = df_stocks[df_stocks['行业'] == ind]
    ax.scatter([ind] * len(subset), subset['价值因子_zscore'],
               alpha=0.4, s=20, color='#e74c3c', label='_nolegend_')
ax.axhline(0, color='black', linestyle='--', linewidth=0.8)
ax.set_title('中性化前 — 行业间存在系统性偏差', fontsize=12, fontweight='bold')
ax.set_ylabel('价值因子 (Z-Score)')
ax.tick_params(axis='x', rotation=45)

ax = axes[1]
for ind in df_stocks['行业'].unique():
    subset = df_stocks[df_stocks['行业'] == ind]
    ax.scatter([ind] * len(subset), subset['价值因子_行业中性'],
               alpha=0.4, s=20, color='#2ecc71', label='_nolegend_')
ax.axhline(0, color='black', linestyle='--', linewidth=0.8)
ax.set_title('中性化后 — 行业偏差已消除', fontsize=12, fontweight='bold')
ax.set_ylabel('价值因子 (行业中性化后)')
ax.tick_params(axis='x', rotation=45)

plt.suptitle('行业中性化效果对比', fontsize=14, fontweight='bold')
plt.tight_layout()
plt.savefig('industry_neutralization.png', dpi=150, bbox_inches='tight')
plt.show()

print("\n行业中性化后，各行业内因子均值为0，跨行业可比性大幅提升。")
print("这是多因子模型中不可或缺的一步，强烈建议所有基于估值的因子都做行业中性化。")
```

## 附录：因子投资的实践建议

### 诚实原则

1. **永远怀疑自己的回测。** 一个在回测中夏普比率 3.0 的策略，实盘中可能只有 0.5。反问自己：有没有用到未来数据？手续费算够了吗？停牌股怎么处理的？样本外测试做了吗？

2. **样本外验证是必须的。** 用 2015-2020 数据训练，用 2021-2023 数据验证——而且验证只能做一次。反复看样本外结果然后回去调参数，样本外也会变成样本内。

3. **理解因果关系，不要只追求相关性。** 雨伞销量和溺水率高度相关，不是因为雨伞导致溺水，而是因为下雨。同理，因子和收益之间如果缺乏经济学逻辑，很可能只是数据挖掘的噪音。

4. **简单优于复杂。** 学术界和业界的大量实践表明，3-8 个逻辑清晰的低频因子组合，往往优于几百个因子构成的复杂黑箱模型。后者几乎一定过拟合。
